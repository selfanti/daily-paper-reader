## 摘要
长时域智能体依赖记忆机制压缩交互历史，但优化记忆写入面临独特的信用分配挑战：一次记忆更新可能因下游工具故障、噪声观测或推理错误而受到奖励或惩罚，而非其自身贡献。这种因果纠缠的信用分配会导致智能体丢弃有用证据或保留无关信息。我们提出HiMPO，一种基于后见之明的记忆策略优化框架，用于在长时域智能体中为记忆写入动作分配低纠缠信用。HiMPO首先通过比较在相同写入前状态下，从先前记忆和更新记忆可恢复的任务相关信息，估计记忆更新的局部效用。然后利用后见相关性作为有界回顾过滤器，当局部效用未得到目标结果支持时，衰减记忆信用。得到的记忆特定优势仅应用于记忆令牌，而轨迹级奖励优化智能体其余行为。在基于评判的开放域任务和客观压缩记忆问答中，HiMPO在保持压缩上下文效率的同时，优于强大的基于记忆和基于强化学习的基线。控制干预进一步表明，HiMPO减少了工具诱导错误导致的责备泄漏，并提高了记忆更新的归因忠实度。

## Abstract
Long-horizon agents rely on memory mechanisms to compress interaction history, but optimizing memory writing faces a distinct credit assignment challenge: a memory update may be rewarded or penalized due to downstream tool failures, noisy observations, or reasoning errors rather than its own contribution. This causally entangled credit can lead agents to discard useful evidence or preserve irrelevant information. We propose HiMPO, a Hindsight-Informed Memory Policy Optimization framework for assigning less-entangled credit to memory-writing actions in long-horizon agents. HiMPO first estimates the local utility of a memory update by comparing the task-relevant information recoverable from the previous and updated memories under the same pre-write state. It then uses hindsight relevance as a bounded retrospective filter that attenuates memory credit when local utility is not supported by the target outcome. The resulting memory-specific advantage is applied only to memory tokens, while trajectory-level rewards optimize the rest of the agent behavior. Across judge-based open-domain tasks and objective compressive-memory QA, HiMPO improves over strong memory-based and RL-based baselines while preserving compressed-context efficiency. Controlled interventions further show that HiMPO reduces blame leakage from tool-induced errors and improves attribution fidelity of memory updates.

# 论文详细总结：HiMPO: 基于后见之明的记忆策略优化，用于长时域智能体中的低纠缠信用分配

## 1. 核心问题与整体含义（研究动机与背景）
- **问题背景**：长时域智能体（如多步搜索、工具增强推理）需要记忆机制来压缩交互历史，避免上下文无限增长。然而，优化记忆写入面临一个特有的**因果纠缠信用分配问题**：最终的成功或失败是记忆更新、工具调用、观测噪声、推理错误等因素共同作用的结果。直接使用轨迹级奖励作为记忆更新的信用信号，会导致工具、观测或推理引入的错误被错误地归因于记忆本身，使智能体可能丢弃有用证据或保留无关信息。
- **核心动机**：为了减少这种纠缠，需要为记忆写入动作分配更忠实、更少干扰的信用。论文提出，记忆信用应基于两个互补的信号：1）**局部反事实效用**：更新记忆相比先前记忆，在相同上下文中能恢复更多任务相关信息；2）**后见相关性**：给定最终成功结果，该记忆写入应变得更具可能性，表明其对结果有特定支持。

## 2. 方法论：核心思想、关键技术细节与算法流程
### 核心思想
HiMPO将每个记忆更新视为策略控制的写入动作，通过**局部反事实效用**（Local Counterfactual Utility）估计其增量价值，并利用**后见相关性**（Hindsight Relevance）作为**有界回溯过滤器**（Bounded Retrospective Filter），在局部效用与后见相关性方向一致时允许信用通过，不一致时将信用衰减。得到的记忆专用优势仅应用于 `<mem>` 令牌，其余令牌使用标准的轨迹级优势。

### 关键技术细节
- **局部反事实效用 (Δ)**：
  - 给定写入前状态 $H_{i,t}$、候选记忆 $m$ 和最终成功结果 $z^*_i$，定义可回答性得分 $S(H_{i,t}, m, z^*_i)$ 为在该状态下用 $m$ 替换记忆后，生成 $z^*_i$ 的平均对数似然。
  - 记忆更新的局部效用为：$\Delta_{i,t} = S(H_{i,t}, m_{i,t}, z^*_i) - S(H_{i,t}, m_{i,t-1}, z^*_i)$，衡量更新带来的边际信息增益。
  - 对 $\Delta_{i,t}$ 在同组内按步骤标准化得到 $\hat{\Delta}_{i,t}$。
- **后见相关性 (log ρ)**：
  - 在给定 $H_{i,t}$ 和目标结果 $z^*_i$ 的条件下，重新计算当前记忆令牌的对数似然 $ \log h_{i,t}$，并在同一提示组和步骤内进行**自归一化**：$\log \rho_{i,t} = \log h_{i,t} - \text{mean}(\log h_{j,t})$。
  - $\log \rho_{i,t} > 0$ 表示该记忆写入在给定目标结果后相对更可能，即与结果相关。
- **门控函数 (G)**：
  - 使用符号一致的门控：$G(\hat{\Delta}_{i,t}, \log \rho_{i,t}) = \sigma(\beta_{\text{eff}} \cdot \text{sgn}(\hat{\Delta}_{i,t}) \log \rho_{i,t} - \tau_\rho)$。
  - 只有当局部效用方向与后见相关性方向一致时，门才开放（$\hat{\Delta}>0$ 且 $\log \rho>0$ 放大正信用；$\hat{\Delta}<0$ 且 $\log \rho<0$ 保留负信用）。方向不一致时，信用被抑制。
- **记忆专用优势**：
  - $A^M_{i,t} = G(\hat{\Delta}_{i,t}, \log \rho_{i,t}) \cdot \hat{\Delta}_{i,t}$。
- **稳定化机制**：
  - **成功轨迹保护掩码**：如果整条轨迹成功（$R_T(\tau_i) \geq \tau_{\text{succ}}$），则将其中的负记忆优势置为零，避免推翻成功证据。
  - **反向平滑**：可选地对记忆优势沿时间步进行指数移动平均，使后期有用的记忆效用向前传播。
- **令牌级优势分配**：
  - 对于 `<mem>` 令牌：$A_{i,k} = A^T_i + \lambda_M \tilde{A}^M_{i,t(k)}$（其中 $\tilde{A}^M$ 是平滑后的记忆优势）；对于其他令牌：$A_{i,k} = A^T_i$。
  - 使用标准的截断策略优化目标（类似 PPO/GRPO）更新策略。

### 算法流程（文字说明）
1. 对每个提示，采样一组轨迹（如 $N=16$）。
2. 计算每条轨迹的轨迹级奖励和优势 $A^T_i$。
3. 对每个记忆写入：
   - 计算局部效用 $\Delta$ 并标准化。
   - 计算后见相关性 $\log \rho$。
   - 通过门控得到 $A^M$。
4. 应用成功轨迹掩码和反向平滑。
5. 将记忆优势分散到 `<mem>` 令牌，其他令牌使用轨迹级优势。
6. 使用截断策略优化目标更新策略网络。
