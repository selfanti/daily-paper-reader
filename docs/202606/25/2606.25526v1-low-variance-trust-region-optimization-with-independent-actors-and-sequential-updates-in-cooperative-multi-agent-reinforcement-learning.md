## 摘要
合作多智能体强化学习假设每个智能体共享相同的奖励函数，并且可以有效地使用单智能体的信任区域框架进行训练。与依赖其他智能体的动作不同，独立执行者设置考虑每个智能体仅基于其局部信息行动，因此具有更灵活的应用。然而，在顺序更新框架中，需要在每个个体智能体的策略步骤后重新估计联合优势函数。尽管重要性抽样在实践中取得了成功，但更新后的优势函数存在指数级高方差问题，这很可能导致不稳定的收敛。在这项工作中，我们首先从经验和理论上分析了高方差优势。为了克服这一限制，我们引入了一个裁剪目标来控制顺序更新中优势波动的上界。通过所提出的目标，我们提供了一个单调界，以次线性收敛到ε-纳什均衡。我们进一步使用我们的裁剪目标推导出了两种新的实用算法。在三个流行的多智能体强化学习基准上的实验结果表明，我们提出的方法在大多数环境中优于测试的基线。通过仔细分析不同的训练设置，我们提出的方法具有稳定的收敛特性和理想的低优势方差估计。为了可重复性，我们的源代码公开在https://github.com/giangbang/Low-Variance-Trust-Region-MARL。

## Abstract
Cooperative multi-agent reinforcement learning assumes each agent shares the same reward function and can be trained effectively using the Trust Region framework of single-agent. Instead of relying on other agents' actions, the independent actors setting considers each agent to act based only on its local information, thus having more flexible applications. However, in the sequential update framework, it is required to re-estimate the joint advantage function after each individual agent's policy step. Despite the practical success of importance sampling, the updated advantage function suffers from exponentially high variance problems, which likely result in unstable convergence. In this work, we first analyze the high variance advantage both empirically and theoretically. To overcome this limitation, we introduce a clipping objective to control the upper bounds of the advantage fluctuation in sequential updates. With the proposed objective, we provide a monotonic bound with sub-linear convergence to $ε$-Nash Equilibria. We further derive two new practical algorithms using our clipping objective. The experiment results on three popular multi-agent reinforcement learning benchmarks show that our proposed method outperforms the tested baselines in most environments. By carefully analyzing different training settings, our proposed method is highlighted with both stable convergence properties and the desired low advantage variance estimation. For reproducibility purposes, our source code is publicly available at https://github.com/giangbang/Low-Variance-Trust-Region-MARL.

- **研究动机**：现有方法（如HAPPO）缺乏对这一高方差问题的系统处理，论文旨在提出一种低方差保证的信任区域优化方法，在保持理论收敛性的同时提升实际训练的稳定性。

## 2. 方法论

- **核心思想**：在顺序更新的优势估计中引入裁剪机制，限制重要性采样比率（ℜ）的上界为1，从而控制方差。
- **关键公式**：
  - 新代理损失函数：
    \[
    L^{1:m, \text{clip}}_\pi(\tilde{\pi}^{1:m-1}, \hat{\pi}^m) = \mathbb{E}_{s\sim d^\pi_\rho, a\sim\pi} \left[ \left( \frac{\hat{\pi}^m(a^m|s)}{\pi^m(a^m|s)} - 1 \right) \min(\mathfrak{R}^{1:m-1}_{\tilde{\pi},\pi}(s,a), 1) A^\pi(s,a) \right]
    \]
  - 其中 \(\mathfrak{R}\) 是前 \(m-1\) 个智能体的联合重要性比率。
- **算法流程**（Algorithm 1）：
  1. 初始化均匀随机联合策略 \(\pi_0\)。
  2. 对于每个迭代 \(k\)：计算优势函数 \(A^{\pi_k}\)；选择步长 \(\beta_k\)；随机排列所有智能体；对每个智能体 \(m\) 依次优化：
     \[
     \pi^{m}_{k+1} \leftarrow \arg\max_{\pi^m} \left[ L^{1:m, \text{clip}}_{\pi_k}(\pi^{1:m-1}_{k+1}, \pi^m) - \beta_k \mathbb{E}^{\rho}_{\pi_k} D_{\text{KL}}(\pi^m \| \pi^m_k) \right]
     \]
  3. 输出随机均匀选出的策略 \(\bar{\pi}\).
- **理论贡献**：
  - 证明单调改进界（Lemma 3），其中包含裁剪带来的额外误差项，但可控。
  - 证明算法以 \(O(1/\sqrt{K})\) 的次线性率收敛到 \(\epsilon\)-纳什均衡（Theorem 4），匹配其他信任区域方法。
  - 方差上界：裁剪后的优势方差被限制在 \(\epsilon^2\) 以内，不再随智能体数量增长。

## 8. 不足与局限

1. **方差分析是悲观上界**：论文承认指数增长是 worst-case 上界，实际中不一定总是出现，裁剪可能在某些情况下过度抑制更新。
2. **探索与方差权衡**：固定裁剪阈值为 1 可能剔除有益的探索信号，特别是在早期训练中高方差可能带来更好的探索。论文未探索自适应裁剪阈值。
3. **全局最优性未证明**：理论仅收敛到纳什均衡，而在独立演员设置下能否达到全局最优仍是开放问题（作者在文中明确提及）。
4. **实验覆盖有限**：未测试大规模多智能体场景（如 SMAC 中更多智能体）或部分可观测性更强的问题（如 LBF）；也未与最新基于 Transformer 的序列模型（如 MAT）对比。
5. **计算代价**：顺序更新在每个迭代中需要依次更新所有智能体，比同时更新（MAPPO）慢；论文未讨论训练时间对比。
6. **偏差风险**：由于独立演员假设，策略无法建模其他智能体的动作，在一些需要精确协作的任务（如对手建模）中可能不足；该方法仅在完全合作奖励函数下验证，未测试混合竞争或零和博弈。
