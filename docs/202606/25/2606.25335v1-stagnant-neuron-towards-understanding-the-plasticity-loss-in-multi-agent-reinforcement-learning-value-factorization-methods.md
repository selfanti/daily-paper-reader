## 摘要
多智能体强化学习（MARL）的值分解方法可能会遭受可塑性损失，在迁移到新任务实例时逐渐难以适应。我们将此问题追溯到停滞神经元，即那些梯度更新相对于权重变得微不足道的单元，从而阻碍学习。尽管存在现有的可塑性注入方法，但事实证明它们对该类神经元无效。为解决此问题，我们提出了知识保持的神经元级可塑性聚焦注入（KNIFE），一种直接针对停滞神经元的新方法。KNIFE将每个停滞神经元替换为一个由三个专门组件组成的复合单元：一个冻结的知识神经元以保留已获取的知识，一个重新初始化的活跃神经元以恢复学习能力，以及一个补偿神经元以确保组合输出与原始输出匹配，从而维持先前学到的协作知识。在SMACv2、捕食者-猎物和矩阵游戏上的大量实验表明，KNIFE显著优于最先进的可塑性注入方法。

## Abstract
Multi-Agent Reinforcement Learning (MARL) value factorization methods can suffer from a loss of plasticity, gradually failing to adapt when transferring to new task instances. We trace this issue to stagnant neurons, units whose gradient updates become negligibly small relative to their weights, thereby hindering learning. While existing plasticity injection methods exist, they prove ineffective for such neurons. To address this, we propose Knowledge-retentive Neuron-level PlastIcity Focusing InjEction (KNIFE), a novel method that directly targets stagnant neurons. KNIFE replaces each stagnant neuron with a composite unit comprising three specialized components: a frozen knowledge neuron to preserve acquired knowledge, a re-initialized active neuron to restore learning capacity, and a compensation neuron to ensure the combined output matches the original, thus maintaining previous learned cooperation knowledge. Extensive experiments on SMACv2, predator-prey, and matrix games demonstrate that KNIFE significantly outperforms state-of-the-art plasticity injection methods.

- **重要性**：现有可塑性注入方法（如ReDo、Plasticity Injection）无法有效处理此类神经元，导致重新适应先前任务时性能严重退化，甚至不如随机初始化的基线。

## 2. 方法论：核心思想与关键技术细节

### 核心思想
提出**KNIFE**（Knowledge-retentive Neuron-level PlastIcity Focusing InjEction），一种在神经元层级直接干预停滞/易变神经元的可塑性注入方法。通过“神经外科手术”将问题神经元替换为一个复合单元，在保持已有协作知识的同时恢复学习能力。

### 关键技术细节

#### 2.1 神经元识别
- 定义**相对更新活跃度（RUA）**：`RUA_i = (mean(||g_i||/||w_i||)) / layer_mean`，其中`g_i`为梯度，`w_i`为权重。
- 停滞神经元：`RUA < α`（默认α=0.25）
- 易变神经元：`RUA > β`（默认β=3）
- 修复集：`S_repair = S_stagnant ∪ S_volatile`

#### 2.2 可塑性注入操作
对每个待修复神经元执行“三神经元替换”：
1. **知识神经元（冻结）**：原样复制原神经元参数并冻结，保留已学知识。
2. **活跃神经元（可训练）**：
   - 输入权重：Kaiming初始化
   - 偏置：初始化为正（`0.5δ_in`），确保立即激活
   - 输出权重：根据下一层权重标准差动态初始化，保证信号兼容
3. **补偿神经元（冻结）**：与活跃神经元共享输入权重和偏置，但输出权重取负，使注入时刻组合输出等于原输出，从而维持知识不变（满足KI++原则）。

#### 2.3 剪枝操作
- 对冻结神经元（知识、补偿）的权重乘以衰减因子γ（0<γ<1），逐渐缩小权重。
- 当权重接近零时剪枝，回收内存。

#### 2.4 理论保证
- 证明KNIFE的注入阶段满足更强的**知识不变性++原则（KI++）**，该原则去除了原KI原则中“仅单个智能体动作不同”的严格限制，适用于更广泛的场景。

## 6. 主要结论与发现

- **停滞神经元是MARL可塑性损失的关键原因**：在任务迁移过程中，停滞神经元集中在混合网络，其比例随着任务数增加而上升，与性能下降呈现强相关。
- **KNIFE显著优于现有方法**：在几乎所有场景下（SMACv2 task-drift/task-switch、缩放、矩阵游戏、捕食者-猎物），KNIFE的胜率/奖励均最高，且能将停滞神经元比率降至最低。
- **KI++原则更优**：满足KI++原则的注入方法（KNIFE）优于仅满足KI原则的方法（ReBorn），因为KI++保证了更广泛的动作偏好不变性。
- **停滞神经元优于休眠神经元作为可塑性指标**：使用停滞神经元作为干预目标比使用休眠神经元带来更大性能提升（图7a）。
- **适用性广**：KNIFE不仅适用于值分解方法，也适用于MADDPG、MAPPO、DGN等非值分解架构。

## 7. 优点

1. **问题发现新颖**：首次在MARL中识别出“停滞神经元”这一病理现象，并建立其与可塑性损失的直接因果关系。
2. **方法设计精巧**：
   - 三神经元复合结构同时保留了知识、恢复了学习能力、保证了输出不变性。
   - 状态感知初始化避免新注入神经元陷入失活区。
   - 剪枝机制控制内存增长。
3. **理论保障充分**：KI++原则提供严格的行动偏好保持保证。
4. **实验验证扎实**：覆盖面广（多种环境、多种任务类型、多种算法），消融实验完整，统计规范。
5. **可迁移性强**：支持多种MARL框架，可做为即插即用模块。

## 8. 不足与局限

1. **缺乏真实世界部署验证**：所有实验仅在模拟基准上完成，未在物理多智能体系统或实际应用中测试。
2. **临时内存开销**：注入阶段会暂时增加参数量，尽管通过剪枝回收，但内存峰值可能高于纯基线。
3. **超参数敏感性尚需探究**：虽然进行了参数敏感性实验（图A15），但α、β、γ等阈值仍依赖经验设定，在更复杂环境中可能需要调整。
4. **跨任务类型泛化**：任务主要涉及单元组成或奖励变化，未涵盖环境动力学本质改变（如物理规则突变）的情况。
5. **理论基础可扩展性**：KI++原则依赖于值函数分解形式，对于非值分解方法（如Actor-Critic）的适用性未作理论分析（尽管实验上有效）。
6. **长期持续学习能力未深入分析**：实验任务序列长度有限（最多11个阶段），更长时间尺度下的可塑性维持尚未评估。
