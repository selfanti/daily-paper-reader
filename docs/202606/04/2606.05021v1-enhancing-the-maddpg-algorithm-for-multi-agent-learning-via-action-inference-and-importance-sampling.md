## 摘要
我们研究了多智能体深度强化学习，并提出了对多智能体深度确定性策略梯度（MADDPG）算法的两项增强。首先，我们引入了一种新颖的动作推断机制，使每个智能体能够预测其他智能体的预期动作，从而提高其自身策略的准确性和稳定性。其次，我们在回放缓冲区中应用了基于几何分布的重要性采样策略，以优先使用更新且信息量更丰富的经验，这有助于缓解多智能体环境中固有的非平稳性。我们在PettingZoo库提供的离散动作捕食者-猎物任务上评估了这两项修改，PettingZoo是一个用于通用多智能体强化学习基准测试的灵活Python接口。我们的结果表明，动作推断能有效提高学习稳定性和智能体间的合作，而使用几何分布的重要性采样相比标准MADDPG可以显著提升探索效率。代码可在https://github.com/shaashwathsivakumar/MARL_Proj获取。

## Abstract
We investigate multi-agent deep reinforcement learning and propose two enhancements to the Multi-Agent Deep Deterministic Policy Gradient (MADDPG) algorithm. First, we introduce a novel Action Inference mechanism that enables each agent to predict other agents' intended actions, thereby improving the accuracy and stability of its own policy. Second, we apply an importance sampling strategy, using geometric distribution, in the replay buffer to prioritize more recent and informative experiences, which helps mitigate the non-stationarity inherent in multi-agent environments. We evaluate both modifications on the discrete-action Predator-Prey task provided by the PettingZoo library, a flexible Python interface for general multi-agent reinforcement learning benchmarks. Our results indicate that Action Inference is effective in improving learning stability and inter-agent cooperation and that importance sampling using geometric distribution can lead to significant improvements in exploration efficiency over standard MADDPG. Code available at https://github.com/shaashwathsivakumar/MARL_Proj

## 1. 核心问题与整体含义（研究动机和背景）
多智能体强化学习（MARL）面临的核心挑战是**环境非平稳性**：每个智能体的策略更新会改变其他智能体所感知的环境动态，导致独立Q学习等方法不稳定。策略梯度方法（如REINFORCE）的方差随智能体数量增长而增大，收敛困难。MADDPG通过集中式训练-分散式执行框架缓解了部分非平稳性，但标准MADDPG仍存在学习效率低、合作不充分的问题。为此，论文提出两项增强：**动作推断机制**和**基于几何分布的重要性采样**，旨在提升学习稳定性和探索效率。

## 2. 方法论：核心思想、关键技术细节

### 2.1 动作推断（Action Inference）
- **核心思想**：每个智能体利用一个预训练的动作推断网络（AI Net）预测其他智能体的上一时刻动作，并将预测结果作为额外输入提供给自己的actor网络，从而提供更丰富的上下文，帮助理解状态转移，实现隐式通信。
- **关键技术细节**：
  - AI Net输入：当前观测 \(o_i\) 和上一时刻观测 \(o_i^{-}\)，计算时间差分 \(\Delta o_i\)。
  - 利用观测的可分解性，设计**定向社会意识模块**（用于预测其他智能体动作）和**定向自我意识模块**（用于预测自身动作），每个模块只处理与单个智能体相关的观测子集，避免 \(O(N^2)\) 的计算复杂度。
  - AI Net通过大量随机交互数据预训练，训练后参数固定，不影响后续MADDPG的损失优化。
- **集成方式**：在MADDPG训练循环中，actor的输入变为 \((o_i, \hat{a}_i)\)，其中\(\hat{a}_i\)来自AI Net的预测。

### 2.2 基于几何分布的重要性采样
- **核心思想**：用几何分布替换均匀采样，优先抽取回放缓冲区中**更近期**的样本，以缓解非平稳性（更近期的数据更反映当前策略）。
- **关键细节**：
  - 几何分布概率质量函数：\(P(X=k) = (1-p)^{k-1}p\)，其中\(k\)为从缓冲区末尾往前数的反向索引。
  - 参数 \(p\) 控制近期偏差强度（论文中取 \(p=10^{-5}\)，缓冲区大小7.5×10^5）。
  - 其他设置不变，仅修改采样方式。

## 6. 主要结论与发现
1. **动作推断（PTAI）** 显著优于标准MADDPG和几何采样，体现在：更早获得高奖励、更平滑的奖励曲线、更强的合作行为。这是因为预测其他智能体动作提供了更丰富的上下文，帮助更快发现高回报策略。
2. **几何分布采样（Geom）** 在大数据量阶段优于标准MADDPG，能偶尔触发政策提升（捕捉到好路径），但稳定性不如PTAI，且后期可能超过PTAI。
3. 标准MADDPG表现最差，收敛慢，奖励波动大。
