## 摘要
现有的记忆增强型LLM智能体仅将过去经验存储在提示空间中，作为文本摘要或检索到的段落，同时在整个 rollout 过程中保持模型参数冻结。此类智能体可以“查找”它们所看到的内容，但无法从中“学习”：其策略不会因经验而改变，任何从上下文中丢弃的信息都会永久丢失。我们引入了TMEM，一个自我进化的参数化记忆框架，其中智能体不仅将历史压缩为显式记忆，还通过轻量级在线更新将提炼的监督信息吸收到快速的LoRA权重Δ_t中，从而在单个 episode 内真正改变其未来行为。我们将其形式化为具有快速权重展开动态的智能体决策过程：动作从π_{θ_0+Δ_t}中采样，而提取动作产生监督信息来更新Δ_t以用于后续决策。这种观点使得提取策略可以直接通过强化学习优化：训练θ_0不仅改进了任务动作，还提高了用于在线LoRA适应的数据质量。我们进一步提出了基于SVD的LoRA子空间初始化以加速在线收敛。在LoCoMo、LongMemEval-S、多目标搜索和CL-Bench上的实验表明，TMEM在不同模型规模上始终优于基于摘要和基于检索的基线方法。

## Abstract
Existing memory-augmented LLM agents store past experience exclusively in prompt space, as textual summaries or retrieved passages, while keeping model parameters frozen throughout a rollout. Such agents can \emph{look up} what they have seen but cannot \emph{learn from} it: their policy is unchanged by experience, and any information dropped from the context is permanently lost. We introduce \texttt{TMEM}, a self-evolving parametric memory framework in which the agent not only compresses history into explicit memory but also absorbs distilled supervision into fast LoRA weights $Δ_t$ via lightweight online updates, genuinely altering its future behavior within a single episode. We formalize this as an agentic decision process with fast-weight rollout dynamics: actions are sampled from $π_{θ_0+Δ_t}$, while extraction actions produce supervision that updates $Δ_t$ for subsequent decisions. This view makes the extraction policy directly optimizable by RL: training $θ_0$ improves not only task actions but also the quality of the data used for online LoRA adaptation. We further propose SVD-based initialization of the LoRA subspace to accelerate online convergence. Experiments on LoCoMo, LongMemEval-S, multi-objective search, and CL-Bench show that \texttt{TMEM} consistently outperforms summary-based and retrieval-based baselines across different model scales.

  - 注意力成本和序列长度呈二次增长，长序列计算昂贵；原始历史含有大量冗余噪音，相关信息淹没其中。

- **动机**：借鉴生物记忆“学习与记忆耦合”的原理，希望LLM智能体能够在**测试时**将有用经验写入自身参数，使记忆不仅占据提示空间，还能**直接改变策略计算**。

- **整体贡献**：提出 **TMEM**（Temporal Memory with Evolving Parameters），一个自我进化的参数化记忆框架，通过在单次展开中执行轻量在线LoRA更新，使得智能体的策略（π_{θ₀+Δₜ}）能够实时进化，同时利用强化学习优化提取策略。

## 2. 方法论：核心思想与技术细节

### 2.1 快速权重展开动态（Fast-Weight Rollout Dynamics）

- 智能体维护三部分状态：
  - **工作上下文** `hₜ`（当前交互文本）
  - **显式记忆** `mₜ`（可选，TMEM中通常为空）
  - **快速参数记忆** `Δₜ`（用LoRA权重表示）

- 动作采样自自适应策略：`aₜ ∼ π_{θ₀+Δₜ}(·|cₜ)`，其中上下文 `cₜ` 可以是 `(q, hₜ, mₜ)`（普通任务/工具动作）或附加记忆写作提示 `d` 的 `(q, hₜ, mₜ, d)`（记忆写动作）。

- 当工作上下文长度超过预算 `L_max` 时，触发**记忆提取模式**：模型生成结构化的QA对（作为监督数据），然后通过轻量在线SFT更新LoRA权重 `Δₜ`。提取后的上下文被清空，后续动作基于更新后的策略。

- 记忆写作提示 `d` 指导模型将当前会话提炼为“指令-输出”格式的QA对，要求内容可回答、简洁、多样化。

### 2.2 参数记忆初始化：SVD 初始化

- 标准LoRA初始化随机子空间，在线TTT步数少时效率低。
- TMEM对FFN投影层使用SVD初始化：对目标权重矩阵 `W = U Σ Vᵀ`，取 top-r 右奇异向量和奇异值，设置：
  - `A₀ = Σ_r V_rᵀ`（冻结），`B₀ = 0`（可训练）。
  - 这样初始增量 `Δ₀ = 0`，但A₀已经张成预训练权重中能量最高的r维子空间。
- 理论定理1证明：在谱对齐假设下，SVD初始化的近似误差≤随机初始化，严格小于当 `ρ_r > r/d_in` 时。

### 2.3 强化学习优化（带停止梯度的策略梯度）

- 将记忆提取视为一个可优化的动作，RL目标优化基模型参数 `θ₀`，使得其既产生好的任务动作，又产生好的提取动作（即能生成高质量的离线SFT监督数据）。
- 使用**停止梯度**（stop-gradient）处理在线更新路径：梯度只回传到策略的对数概率，而不通过LoRA更新操作符 `T` 反向传播。

## 6. 主要结论与发现

- TMEM在**所有4个基准**上，无论是否经过RL训练，均一致优于无记忆、摘要式和检索式基线。
- **在线参数记忆**（快速LoRA权重）能够比纯文本记忆更有效地保留和利用经验，尤其适用于需要精确事实回忆的任务（如LongMemEval-S，提升+10.21 F1 / +7.02 EM）。

- **SVD初始化**相比随机初始化显著提升EM（+3点以上），证明锚定高能量子空间对少量步在线更新有利。
- **QA对形式**作为监督信号优于摘要和下一词预测，尤其在精确定位事实时。
- 效率方面TMEM介于摘要法和检索法之间，但能带来显著的准确率提升，属于可接受的trade-off。
