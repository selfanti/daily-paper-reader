## 摘要
语言体面临的一个核心挑战是如何利用过往经验来适应动态的测试时条件。尽管近期研究展示了体记忆机制的前景，但大多数系统将检索限制在情节启动阶段。因此，随着长期任务逐步展开，体被迫依赖静态指导，这些指导会变得越来越不匹配。为解决这一僵化问题，我们提出了自适应记忆体（AdaMEM），一种用于体测试时自适应的新型框架。在不更新模型在线参数的情况下，AdaMEM通过一种混合记忆架构来适应体行为：它维护了从离线收集的原始经验中获取的长程轨迹记忆，同时动态生成短程策略记忆以实时指导决策。这种机制能够在不同的推理时计算水平下权衡令牌效率与适应性。实验表明，AdaMEM显著优于静态记忆基线，在ALFWorld上相对提升高达13%，在WebShop上达到11%，并在HotpotQA上的体搜索中持续保持领先性能。为进一步增强这种适应性，我们开发了STEP-MFT，一种逐步记忆微调技术，训练策略从检索到的经验中合成高质量策略，从而获得额外性能提升。我们的工作为体记忆建立了新的扩展维度，支持在真实世界环境中部署后的持续推理与自我进化。代码已开源：https://github.com/yunx-z/AdaMEM。

## Abstract
A central challenge for language agents is utilizing past experience to adapt to dynamic test-time conditions. While recent work demonstrates the promise of agentic memory mechanisms, most systems restrict retrieval to episode initiation. Consequently, agents are forced to rely on static guidance that becomes increasingly misaligned as long-horizon tasks unfold. To address this rigidity, we propose the Adaptive Memory Agent (AdaMEM), a novel framework for agent test-time adaptation. Without updating model parameters online, AdaMEM adapts agent behavior via a hybrid memory architecture: it maintains a long-term trajectory memory of raw experiences collected offline while generating dynamic short-term strategy memory on-the-fly to guide decision-making. This mechanism enables the trade-off between token efficiency and adaptability across varying inference-time compute levels. Empirically, AdaMEM significantly outperforms static memory baselines, achieving relative gains of up to 13% on ALFWorld and 11% on WebShop, with consistent leading performance extending to agentic search on HotpotQA. To further enhance this adaptation, we develop STEP-MFT, a Step-wise Memory Fine-Tuning technique that trains the policy to synthesize high-quality strategies from retrieved experiences, yielding additional performance gains. Our work establishes a new scaling dimension for agentic memory, supporting continuous reasoning and self-evolution post-deployment in real-world environments. Our code is available at https://github.com/yunx-z/AdaMEM.

语言体（Language Agents）在长程任务（如网页导航、具身操作）中依赖过往经验进行决策。现有记忆机制（如Synapse、ReasoningBank）仅在**情节启动时**进行一次性检索，导致后续决策依赖静态引导，无法适应中间失败或子目标变化。这种**静态检索的僵化**使得体在动态测试时环境中性能下降。本文旨在解决如何实现**测试时连续自适应**，而无需在线更新模型参数。

## 2. 方法论：核心思想与关键技术细节
- **混合记忆架构（Hybrid Memory）**：
  - **长期轨迹记忆（Long-Term Trajectory Memory, M）**：离线收集的成功轨迹，按状态嵌入索引，存储（状态, 动作, 剩余轨迹）键值对。
  - **短期策略记忆（Short-Term Strategy Memory, zt）**：在线根据当前状态st检索相关经验，动态合成简洁的自然语言策略，指导下一步动作。
- **两种推理模式**：
  - **AdaMEM-Low**：低适应努力，策略持久化，仅在必要时（检测到策略过时）触发刷新（drefresh=yes），节省令牌。
  - **AdaMEM-High**：高适应努力，每一步都生成临时策略，丢弃后立即失效，最大化适应性。
- **STEP-MFT（Step-wise Memory Fine-Tuning）**：
  - 一种**过程级拒绝采样微调**技术，用于训练体生成高质量策略。
  - **过滤条件**：只保留**成功轨迹**中**策略改变了动作**（at ≠ a′t，a′t 为无记忆基线动作）的样本，确保策略具有**正优势（Positive Advantage）**。
  - 训练目标：监督微调（SFT）最大化策略生成概率 `log πθ(z* | s, Eret)`。
- **理论支持**：证明了**动作改变**是策略优势为正的必要条件（Proposition 3.1）。

- **混合记忆支持跨模型泛化**（Gemma使用Qwen记忆仍有效）。

- AdaMEM建立了**效率-性能的帕累托前沿**，将额外测试时计算转化为性能提升。

## 7. 优点
- **新颖的混合记忆架构**：解耦长期存储与短期抽象，实现测试时零参数自适应。
- **过程级奖励无需额外rollout**：通过简单的动作比较进行拒绝采样，成本低且有效地甄别高价值策略。
- **跨模型泛化能力**：长程记忆可由不同模型共享，短期策略由当前模型生成，避免了策略不兼容。
- **可扩展性好**：检索更多经验时性能单调提升，不受上下文长度限制。
- **实际部署友好**：提供两种推理模式（Low/High）以灵活平衡效率与适应性，且微调后统一模型对策略和动作生成，简化部署。

## 8. 不足与局限
- **策略惰性（Strategy Inertia）**：AdaMEM-Low依赖模型自我评估是否刷新策略，当模型过度自信时可能拒绝刷新，陷入死循环（如案例2）。

- **环境假设**：实验仅在纯文本环境，未验证在多模态或真实物理场景下的表现。
- **计算开销**：虽然比Synapse更低，但仍需策略合成和检索，计算成本高于无记忆基线。
- **微调数据规模有限**：仅使用1 epoch的SFT，未探索RL或更大规模训练是否能带来更显著提升。
- **未与最新端到端RL记忆方法（如MemRL）直接比较**，可能低估了其他动态优化方法的潜力。
