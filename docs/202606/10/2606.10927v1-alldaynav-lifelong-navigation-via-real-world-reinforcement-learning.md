## 摘要
动态环境下的终身具身导航要求机器人从碎片化观测中形成持久场景理解，这对依赖显式地图或场景图的现有方法而言仍具挑战，且这些方法难以泛化到非结构化场景。我们提出AllDayNav，一种终身自学习导航框架，通过强化学习将场景动态隐式编码到大模型十亿级参数中，并由自进化多模态记忆驱动——该记忆维护并更新视觉关键帧、语义描述及时间上下文，同时自主生成开放词汇指令、图像目标与结构化奖励。在跨房间、跨回合及跨任务场景下的合成与真实环境实验中，AllDayNav的成功率接近100%，且在路径效率与鲁棒性上持续超越基于地图、VLM及RL的强基线方法，证明了隐式记忆驱动的强化学习可作为显式地图构建的可扩展替代方案，用于可靠的终身导航。

## Abstract
Lifelong embodied navigation in dynamic environments requires robots to form persistent scene understanding from fragmentary observations, which remains difficult for existing methods that rely on explicit maps or scene graphs and struggle to generalize beyond structured settings. We propose AllDayNav, a lifelong self-learning navigation framework that implicitly encodes scene dynamics into the billion-scale parameters of a large model via reinforcement learning, powered by a self-evolving multimodal memory that maintains and updates visual keyframes, semantic descriptions, and temporal context while autonomously generating open-vocabulary instructions, image goals, and structured rewards. Experiments in both synthetic and real-world environments across cross-room, cross-episode, and cross-task scenarios show that AllDayNav achieves success rates approaching $100\%$ and consistently surpasses strong map-based, VLM, and RL baselines in path efficiency and robustness, demonstrating implicit, memory-driven reinforcement learning as a scalable alternative to explicit mapping for reliable lifelong navigation.

## 二、论文提出的方法论：核心思想、关键技术细节

- **核心思想**：AllDayNav 通过一个自进化的多模态记忆数据库与 VLA 策略的协同进化循环，实现终身自学习。记忆存储视觉关键帧、语义描述和时间戳；系统自主生成“指令-目标图像”训练对，并通过保守Q学习（CQL）在线优化策略，无需人工标注。
- **关键技术细节**：
  1. **自进化多模态记忆数据库（Memory Database M）**  
     - 每条记录：(图像oi, 文字描述di, 时间戳τi, 嵌入向量ei)。  
     - 构建：探索中每5步取候选关键帧，通过视觉嵌入余弦距离（阈值0.15）筛选；再用全局去重（阈值0.7）决定插入或更新（对齐度更高时替换）。  
     - VLM（Qwen3-VL-8B-Instruct）生成描述、执行检索（包括时空指令）。
  2. **自我指令生成与混合目标检索**  
     - 每10回合或记忆增长20%时，VLM依据记忆中的描述生成50条多样性指令。  
     - 混合检索：优先从记忆检索（置信度需≥0.6），否则fallback到互联网图像库Ginternet，早期探索时fallback概率随记忆稀疏度增加。
  3. **预训练 VLA 导航骨干**  
     - 双编码器：SigLIP（语义）+ DINOv2（结构），融合后经MLP投影。  
     - 历史观测通过全局平均池化压缩，并施加指数衰减注意力偏置（λ控制），窗口64帧。  
     - 动作空间：8个未来航点 (Δx,Δy,Δθ)。  
     - 预训练：Habitat模拟器，50,000条轨迹（80%最短路径专家轨迹+20%随机探索轨迹），总500万步，使用 MSE 损失。
  4. **保守 Q 学习（CQL）在线优化**  
     - 奖励函数：终端成功+10，阶段进展（相似度0.5/0.6/0.7首次突破+2），时间惩罚-0.01，碰撞-0.1。  
     - 使用LLaVE-7B异步计算视觉相似度作为奖励基础。  
     - CQL 损失：标准 Bellman 误差 + 保守正则项（α=5.0）。  
     - 策略损失：Q值最大化 + 行为克隆正则化（λ_bc=0.1）。  
     - 更新频率：每100步更新一次。
  5. **Sim-to-Real 增强**：预训练时对当前帧和目标帧独立施加随机模糊、抖动、裁剪等增强，模拟真实域差异。
