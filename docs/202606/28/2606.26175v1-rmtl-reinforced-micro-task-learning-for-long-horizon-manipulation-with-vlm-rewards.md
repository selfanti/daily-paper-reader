## 摘要
强化学习在机器人操作中通常需要手动设计稠密的奖励函数，这难以调整且往往脆弱，或者需要从人类演示或偏好中学习奖励，这成本高昂。最近的一系列工作使用预训练的视觉语言模型（VLM）作为零样本奖励模型，用单个文本提示替代了这些成本。然而，我们认为对于具有随机初始条件的长时序操作任务，单个全局提示过于粗糙。单提示VLM奖励在轨迹的大部分区域接近平坦，使得代理难以检测早期进展。我们提出强化微观任务学习（RMTL），该方法将操作任务分解为一小组由语言描述的微观任务，并训练代理在这些任务之间切换。每一步，代理都会收到一个多视角VLM奖励，该奖励使用当前活跃微观任务的提示计算，并在多个相机视角上平均以减少视图特定遮挡的影响。反向课程逐渐将代理暴露于更困难的初始条件，而PPO工作器首先使用固定的基于距离的规则进行训练，该规则选择活跃的微观任务。然后，我们用学习到的层次管理器替换此规则，将基于规则的阶段选择转变为完全学习的层次策略。我们在Fetch操作环境中使用三个短阶段特定提示实例化RMTL，无需额外的提示调整。实验表明，RMTL提供了比单提示VLM奖励更信息丰富的奖励信号，从而加速学习。这些结果表明，将VLM奖励分解为微观任务特定语言提示可以显著提高语言引导的强化学习在机器人操作中的可扩展性。

## Abstract
Reinforcement learning (RL) for robotic manipulation often requires manually designing a dense reward function, which is difficult to tune and often fragile, or learning a reward from human demonstrations or preferences, which can be expensive. A recent line of work uses pretrained vision-language models (VLMs) as zero-shot reward models, replacing these costs with a single text prompt. However, we argue that a single global prompt is too coarse for long-horizon manipulation tasks with randomized initial conditions. The single-prompt VLM reward is near-flat for much of the trajectory, making early progress hard for the agent to detect. We propose Reinforced Micro-Task Learning (RMTL), an approach that decomposes a manipulation task into a small set of language-described micro-tasks and trains the agent to switch between them. At each step, the agent receives a multi-view VLM reward computed using the prompt of the currently active micro-task and averaged across multiple camera views to reduce the effect of view-specific occlusions. A reverse curriculum gradually exposes the agent to harder initial conditions, while a PPO worker is first trained with a fixed distance-based rule that selects the active micro-task. We then replace this rule with a learned hierarchical manager, turning rule-based phase selection into a fully learned hierarchical policy. We instantiate RMTL on the Fetch manipulation environment using three short stage-specific prompts and without additional prompt tuning. Experiments show that RMTL provides more informative reward signals than single-prompt VLM rewards, enabling faster learning. These results suggest that decomposing VLM rewards into micro-task-specific language prompts can substantially improve the scalability of language-guided reinforcement learning for robotic manipulation.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：在长时序机器人操作任务中，使用预训练视觉语言模型（VLM）作为零样本奖励模型时，单个全局提示（如“机器人夹爪举起红色方块”）会产生近乎平坦的奖励信号。由于前期状态与最终目标的语义相似度低，奖励几乎不变，导致代理难以检测早期进展，无法通过强化学习有效引导探索。
- **背景**：传统强化学习要么依赖手工设计的稠密奖励（脆弱、难迁移），要么需要人类演示或偏好学习（成本高、难扩展）。VLM奖励模型通过一个文本提示替代了这些成本，但单提示策略在长时序、随机初始条件下失效。
- **整体含义**：作者提出通过任务分解为多个语言描述的微任务，每个微任务使用独立的VLM提示，从而将平坦的奖励变为分段单调信号，使PPO能够有效学习。这一方法显著提升了语言引导强化学习在机器人操作中的可扩展性。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- **微任务分解**：将长时序操作任务（如抓取并举起方块）分解为三个语言描述的微任务：`approach`（靠近）、`align`（对齐）和`grasp`（抓取）。每个微任务对应一个短提示，在对应阶段使用该微任务的提示计算VLM奖励。
- **多视图聚合**：从6个固定相机视角渲染图像，每个视角独立计算VLM分数，取算术平均作为最终奖励，以减少单视图遮挡噪声。
- **反向课程学习**：从简单初始条件（夹爪靠近物体）逐渐过渡到完全随机初始条件，共5个难度等级，根据在线成功率自动晋级。
- **层次化学习**：第一阶段使用基于距离的规则选择活跃微任务，训练PPO工作器；第二阶段用学习到的层次管理器（一个小型MLP）替换规则，通过行为克隆（BC）预热后使用REINFORCE优化，最终实现完全的层次化策略。

### 关键技术细节
- **VLM评分器**：采用CLIP风格的投影评分器（Rocamonde et al., 2024），使用基线提示（“桌子上有红色方块和机器人夹爪”）和微任务目标提示，计算图像嵌入与目标提示沿“基线→目标”方向的投影距离，公式：  
  `r(i)_v = 1 - 1/2 * || (x_v - e_τi) · P_α(d_i) ||^2`  
  其中 `d_i = e_τi - e_b`，`P_α` 是投影混合矩阵（α=0.9）。多视图平均后得到`r(i)_t`。
- **奖励塑形**：对原始VLM相似度进行在线自适应归一化（观察到的min/max），并加入下降惩罚，输出范围[-1,0]，成功时额外+1。
- **管理器训练**：管理器每H=3步决策一次，输入状态（25维），输出3类微任务。先离线BC从规则标签学习3500步，再使用REINFORCE优化，包含KL散度锚向规则策略和熵正则化。工作器前50000步冻结actor，之后以较低学习率解冻。
- **规则选择器**：基于夹爪-方块距离的阈值：距离>0.15m→approach，[0.04,0.15]→align，<0.04m→grasp。

## 6. 主要结论与发现

- 将VLM奖励分解为三个微任务提示，将近乎平坦的单提示奖励变为分段单调信号，使PPO能成功学习（其他方法失败）。
- 多视图聚合显著降低单视角遮挡噪声，奖励曲线更平滑。
- 反向课程是必要组件：没有它，随机初始化下VLM奖励不足引导探索。
- 学习型层次管理器（Stage 2）在随机初始化下达到~98%成功率，优于规则基（94%）和非层次PPO（89%），而单提示PPO为0%。
- 学习管理器与规则选择器有约73%的决策一致性，但性能更高，表明规则提供了良好归纳偏置。

## 7. 优点（亮点）

- **方法创新**：将VLM奖励与微任务分解、多视图聚合、反向课程、层次化学习有机结合，实际解决了单提示奖励平坦的问题。
- **可扩展性**：仅需少量语言提示（3个），无需额外提示调优，即可显著提升学习效果。
- **自动化**：第二阶段用学习管理器取代手工规则，迈向完全自主学习。
- **实验严谨**：对每个核心组件进行了消融，提供定性（奖励曲线）和定量（成功率）双重证据。
- **实用性强**：方法仅依赖视觉观察（无需特权状态），适用于真实机器人场景（渲染只是辅助，原理可迁移）。

## 8. 不足与局限

- **实验覆盖有限**：仅在一个环境（FetchPickAndPlace）和一个任务上验证，未与其他基准（如VIP、LIV、Eureka等）在更多任务上对比，泛化性存疑。
- **VLM依赖**：奖励信号继承VLM的感知局限（有限的空间推理、对光照/纹理敏感）。在接触密集阶段可能噪声大或饱和。
- **手工元素**：微任务提示和阶段数量（3个）人工设定，未自动化。未来可结合LLM自动分解。
- **训练成本**：Stage 2需要先冻结工作器再解冻，流程较复杂；且两阶段训练总步数达800k，仍然不低。
- **奖励饱和**：抓取提示在成功抓取后饱和，导致目标附近梯度弱，需额外塑形处理。
- **缺乏真实机器人实验**：仅在仿真中测试，未考虑领域迁移到真实环境的挑战。
