## 摘要
我们提出了带奖励机器的帕累托Q学习（PQLRM），这是一种适用于奖励结构由一组奖励机器（RM）指定的任务的多目标强化学习算法。PQLRM结合了帕累托Q学习（PQL），该学习通过维护向量值Q估计集来近似帕累托前沿，并利用了来自带奖励机器的Q学习（QRM）的增强功能，后者利用了奖励信号的因子自动机结构。这产生了一种多策略算法，在非马尔可夫、RM编码的奖励下仍保持样本效率。实验表明，PQLRM比应用于叉积MDP的朴素PQL基准收敛更快，并且能够合成QRM无法获得的帕累托最优策略。

## Abstract
We present Pareto Q-Learning with Reward Machines (PQLRM), a multi-objective reinforcement learning algorithm for tasks whose reward structure is specified by a set of reward machines (RMs). PQLRM combines Pareto Q-Learning (PQL), which maintains sets of vector-valued Q-estimates to approximate the Pareto front, with enhancements from Q-Learning with Reward Machines (QRM), which exploits the factored automaton structure of the reward signal. This yields a multi-policy algorithm that remains sample-efficient under non-Markovian, RM-encoded rewards. Experimental trials show that PQLRM converges faster than a naive PQL baseline applied to the cross-product MDP and can synthesize Pareto-optimal policies that QRM cannot.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **研究动机**：现实世界的强化学习任务通常需要平衡多个相互冲突的目标（如性能、安全、能耗），即多目标优化问题。传统单标量奖励的RL方法无法直接处理。同时，许多任务的奖励结构具有非马尔可夫性（例如依赖于历史动作或状态序列），而奖励机器（Reward Machines, RMs）提供了一种结构化的自动机表示来处理此类奖励。
- **核心问题**：现有方法在非马尔可夫、RM编码的多目标奖励场景下样本效率低下，且难以同时学习一组帕累托最优策略。具体而言：帕累托Q学习（PQL）直接处理叉积状态空间（环境状态×RM状态）导致维度过高；而带奖励机器的Q学习（QRM）仅能学习单目标最优策略，无法获得多目标权衡下的完整帕累托前沿。
- **整体意义**：本文提出PQLRM，融合PQL和QRM的优点，既能利用RM的结构分解加速学习，又能输出一组帕累托最优策略，提升了复杂奖励结构下的样本效率和策略多样性。

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程
- **核心思想**：将PQL的向量值Q估计集合与QRM的联合RM状态更新机制结合。为每个联合RM状态（各个RM的状态组合）维护独立的Q向量集合，利用单个环境转移同时更新所有RM状态下的Q集，从而大幅提升样本效率。
- **关键技术细节**：
  - 定义多目标MDP与RM的组合模型MOMDPRM，其中包含d个RM，每个RM有自己的状态机。
  - 对每个(环境状态s, 动作a)及每个联合RM状态⃗v，维护一个Q向量集合Q_set(s, a)（初始为空），以及平均即时奖励向量ℛ(s,a)和未来非支配向量集合ND(s,a)。
  - 更新时，从环境观察一个转移(s,a,s′)，遍历所有可能的联合RM状态⃗v，计算对应的下一联合状态⃗v′=δ_st(⃗v, L(s′))，合成所有RM的即时奖励向量r。然后更新ND和ℛ，最后通过“向量-集合加法”重构Q_set。
  - 动作选择：采用基于超体积或帕累托基数ε-贪心策略。
  - 策略重构：在初始状态选择一个目标Q向量，然后在每一步根据当前联合RM状态选择使剩余目标接近的动作。
- **算法流程**（文字说明）：
  1. 初始化所有状态-动作-联合RM状态对的Q_set为空。
  2. 循环每个episode：
     - 重置环境状态s和实际联合RM状态⃗u。
     - 循环直到终止：
       - 根据当前s和⃗u，从Q_set选择动作a。
       - 执行a，得到s′和标签σ。
       - 遍历所有联合RM状态⃗v：
         - 计算⃗v′和合成奖励向量r。
         - 更新ND和ℛ。
       - 推进s←s′，⃗u←δ_st(⃗u,σ)。
  3. 输出所有Q_set。

## 3. 实验设计：数据集/场景、基准、对比方法
- **实验场景**：
  - **PBST (Pressurized Bountiful Sea Treasure)**：一个3×11网格世界，潜艇收集宝藏，三个目标：最小化时间（每步-1）、最大化宝藏价值、最小化深度压力（非马尔可夫，需RM）。共有15个帕累托最优策略。
  - **Office World**：一个办公室地图，包含咖啡、邮件、装饰等地点。共三个子实验：
    - Exp2：三个任务（到达办公室、取咖啡、取邮件），需尽量少步数，6个帕累托最优向量。
    - Exp3：三个任务（到达办公室、不碰装饰取咖啡、不碰装饰取邮件），4个帕累托最优策略。
    - Exp4：两个任务（巡逻A→B→C→D后到办公室、不碰装饰到办公室），2个帕累托最优策略。
- **对比方法**：
  - **PQL（基线）**：将RM状态折叠到环境状态，运行标准帕累托Q学习。
  - **QRM**：只能学习单目标最优策略，报告各策略期望奖励之和的归一化值。
- **评价指标**：对于多策略算法（PQLRM和PQL）报告归一化超体积；对于QRM报告归一化期望奖励之和。每2000步评估一次。

## 6. 论文的主要结论与发现
- PQLRM在样本效率上显著优于朴素PQL基线：在PBST中收敛速度稍快，在Office World的Exp2、Exp3中PQLRM更快达到更高超体积，在Exp4中PQL只有零超体积（无法完成任务），而PQLRM成功。

- 奖励机器的结构分解对多目标学习有益：利用单个转移更新所有RM状态，避免叉积状态空间带来的复杂度。
