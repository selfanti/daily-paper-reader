## 摘要
对话式AI智能体需要既具备可扩展性又在长时间交互跨度内保持语义连贯的记忆系统。现有方法主要依赖基于大语言模型（LLM）的写入时摘要，这引入了非确定性、不断增长的令牌成本以及修剪决策的不透明性。我们提出确定性记忆框架（DMF），一种以CPU优先的方法，用完全确定性的流水线替代生成式记忆压缩，该流水线基于经典NLP分析、向量几何和数学评分。DMF为每次对话交互分配一个生存分数Ω，该分数由确定性内容信号、对话线索和结构化来源通过逻辑投影组合计算得出。一种交互计数衰减律（记为Ω_eff(Δn)）控制着随着新轮次到来相关性的演变，其中Δn是更新交互的数量而非实际时间，从而保持完全确定性。我们给出了DMF的数学公式、结构化召回流水线、修剪决策过程以及评估协议。实验在使用LoCoMo和LongMemEval数据集专门构建的基准上进行。我们将DMF与Mem0（一种流行的AI智能体记忆层）进行比较。DMF在准备记忆上下文时使用零令牌，在整段对话中使用的令牌数量减少5倍至242倍，同时达到相当的准确率。这些结果表明，可以从记忆管理循环中消除LLM调用，将令牌成本降至接近零，并为对话式AI智能体实现确定性记忆系统。

## Abstract
Conversational AI agents require memory systems that are both scalable and semantically coherent across long interaction horizons. Existing approaches rely predominantly on large language model (LLM)-based summarisation at write time, which introduces non-determinism, escalating token costs, and opacity in pruning decisions. We present the Deterministic Memory Framework (DMF), a CPU-first approach that replaces generative memory compression with a fully deterministic pipeline grounded in classical NLP analysis, vector geometry, and mathematical scoring. DMF assigns each conversational interaction a Survival Score $Ω$ computed from deterministic content signals, conversational cues, and structured provenance, combined through a logistic projection. An interaction-count decay law, denoted as $Ω_{\mathrm{eff}}(Δn)$, governs how relevance evolves as new turns arrive, where $Δn$ is the number of newer interactions rather than wall-clock time, preserving full determinism. We present the mathematical formulation of DMF, its structured recall pipeline, the pruning decision procedure, and the evaluation protocol. Experiments are conducted on a purpose-built benchmark using the LoCoMo and LongMemEval datasets. We compare DMF against Mem0, a popular memory layer for AI agents. DMF achieves comparable accuracy while using zero tokens to prepare the memory context and 5x to 242x fewer tokens over the entire conversation. These results show that it is possible to eliminate LLM calls from the memory-management loop, reducing token costs to nearly zero and enabling deterministic memory systems for conversational AI agents.

## 1. 论文的核心问题与整体含义（研究动机和背景）

现有对话式AI智能体的记忆系统主要依赖大语言模型（LLM）在写入时进行摘要压缩，这存在四个关键缺陷：
- **非确定性**：相同历史在不同LLM调用下可能产生不同摘要，导致记忆状态不可复现，难以调试和审计。
- **高令牌成本**：每次摘要调用消耗与历史长度成比例的输入令牌，并产生输出令牌，长期运行成本极高。
- **语义漂移**：摘要一旦写入便固定，后续NLP模型或对话语境变化时，摘要仍锚定于旧解释。
- **不透明性**：无法审计某条事实为何被保留或删除，因为决策深埋于神经生成步骤中。

因此，论文提出完全确定性的记忆框架**DMF（Deterministic Memory Framework）**，在不调用任何LLM的情况下管理对话记忆，实现零令牌成本、可复现、可审计的记忆系统。

## 2. 论文提出的方法论

### 核心思想
用**确定性管道**替代生成式记忆压缩：基于经典NLP分析、向量几何和数学评分，为每次交互计算**生存分数Ω**，并通过**交互计数衰减律**（而非实际时间）控制相关性演变。所有操作（评分、修剪、归档、检索、重排序、证据组装）均无需生成式模型调用。

### 关键技术细节
- **生存分数Ω**：由三个线性通道通过逻辑斯蒂函数投影得到：
  - 内容通道：信息密度ID、情感强度|S|、归一化实体数E_norm、语义散度D（负惩罚）。
  - 操作通道：约束、偏好、当前状态、纠正、替换等对话信号（加性贡献）。
  - 来源通道：用户纠正、偏好更新、约束来源等结构化元数据。
  综合预激活值 `z_total = z_content + z_op + z_prov`，然后代入sigmoid：`Ω = σ(z_total - x0)`，映射到(0,1)。
- **交互计数衰减律**（式18）：
  `Ω_eff(Δn) = Ω * exp(-λ * (1 - ηΩ) * Δn)`
  其中Δn是新交互的数量（而非时间），λ是基础衰减率，η∈[0,1)是惯性强度的超参数。高Ω的条目衰减更慢（得分依赖性惯性）。
- **修剪机制**：
  - **硬清除**：当Ω_eff < Ω_kill（默认0.05）时直接删除。
  - **预算压力修剪**：当活动队列令牌超预算时，仅允许Ω_eff ≤ τ_H的条目参与，并计算修剪分数`S_prune = Ω_eff + 保留奖励 - 主题取代惩罚`，选取最小值进行删除。
- **长期记忆（LTM）归档**：只保存原始文本记录和来源，不保存分数、嵌入等派生字段，保持稳定。可选添加确定性卡片投影（subject-predicate-object事实）辅助检索。
- **召回管道**：查询理解 → 多通道候选生成（原始语义检索、词汇检索、卡片语义/符号检索） → 可回答性重排序 → 证据组装 → 上下文渲染。在提示中先呈现LTM召回证据，后呈现活动对话。

### 公式与算法流程（文字说明）
1. 每轮新交互：提取NLP信号（ID, |S|, E, D）和对话信号。
2. 计算Ω（结合内容、操作、来源）。
3. 将条目加入活动队列，应用交互计数衰减更新所有Ω_eff。
4. 定期执行硬清除，检查令牌预算并执行预算压力修剪。
5. 查询时：解析查询，从LTM和活动记忆检索候选，硬过滤，可回答性重排序，组装证据，渲染到提示中。

## 7. 优点

- **完全确定性**：记忆状态唯一取决于算法和配置，不受LLM采样随机性影响，便于科学基准测试和审计。
- **极低的令牌消耗**：记忆管理阶段零LLM调用，大幅降低运营成本（尤其在长对话中）。
- **结构化召回流水线**：结合原始文本检索、卡片符号检索、可回答性重排序、证据组装，提高召回质量和可解释性。
- **交互计数衰减**：以新交互数量而非实际时间为轴，使记忆状态成为消息序列的确定性函数，避免因暂停导致的不公平衰减。
- **评分依赖性惯性**：高质量条目（高Ω）衰减更慢，维持相对优势，接近心理记忆模型。
- **源规范记忆设计**：LTM只存原始文本，派生卡片仅作辅助，保证召回随管道演变而不污染源数据。
- **CPU友好**：无需GPU即可部署，适合低成本场景。
