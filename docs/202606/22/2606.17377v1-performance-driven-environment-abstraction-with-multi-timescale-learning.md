## 摘要
我们研究面向大马尔可夫决策过程中的决策问题的性能驱动环境抽象。与保持几何或拓扑结构不同，我们寻求直接优化决策质量的抽象。我们将抽象建模为一种受控近似，通过聚合状态空间并在每个聚合状态内强制执行共享动作分布来实现。对于固定划分，我们建立了一个性能保证，将价值函数近似误差与动作共享引入的损失分离开来。在此分析的指导下，我们开发了一个多时间尺度强化学习框架，该框架联合调整策略和树形结构环境抽象。该算法根据Q值差异细化和粗化状态空间区域，在性能与抽象大小和复杂性之间取得平衡。实证结果表明，与演员-评论家基线方法相比，该方法实现了显著的状态压缩、更高的样本效率和更快的重新规划。

## Abstract
We study performance-driven environment abstraction for decision-making in large Markov decision processes. Rather than preserving geometric or topological structure, we seek abstractions that directly optimize decision quality. We model abstraction as a controlled approximation obtained by aggregating the state space and enforcing a shared action distribution within each aggregated state. For a fixed partition, we establish a performance guarantee that separates value-function approximation error from the loss introduced by action sharing. Guided by this analysis, we develop a multi-timescale reinforcement learning framework that jointly adapts the policy and a tree-structured environment abstraction. The resulting algorithm refines and coarsens regions of the state space based on Q-value discrepancies, balancing performance against abstraction size and complexity. Empirical results demonstrate substantial state compression, improved sample efficiency, and faster replanning compared to actor-critic baselines.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **研究动机**：在大规模马尔可夫决策过程（MDP）中，直接在全状态空间进行规划和求解往往计算不可行，且无法满足安全关键任务的实时性要求。传统环境抽象方法（如信息论、拓扑结构启发式）保留几何或拓扑结构，但**不直接优化决策质量**，导致抽象与任务性能脱节。
- **核心问题**：如何构建**性能驱动**的状态抽象，即直接优化决策质量（而非保真几何/拓扑），在压缩状态空间的同时最小化性能损失。
- **背景**：人类通过分层表示处理复杂任务，在粗粒度上决策、必要时精细推理。本文希望自动在线构建这种层级抽象，使智能体能在样本效率和决策质量之间自适应平衡。

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：将环境抽象建模为**受控的状态聚合**，每个聚合状态（superstate）内强制共享动作分布（Same-Action-Distribution, SAD约束）。通过联合优化策略和树形抽象（quadtree），实现自适应细化与粗化。
- **关键技术细节**：
  - **性能分解定理（Theorem 2）**：对固定聚合Γ，推导出性能损失上界：  
    `∥ΨΓ̄V^*_{Γ,μ^*} - V^*∥∞ ≤ (2ϵΓ + δΓ)/(1-β)`  
    其中ϵΓ是值函数近似误差（由值函数在superstate内波动引起），δΓ是SAD约束引起的动作共享误差（同一superstate内所有状态被迫选择相同动作）。
  - **可计算上界（Corollary 3）**：用贝尔曼残差不匹配替换上述误差，得到`̄ϵΓ = ∥ΨΓBΓ,μ^*̄V^*_{Γ,μ^*} - B^SΨΓ̄V^*_{Γ,μ^*}∥∞`，可直接通过Q估计近似。
  - **树形抽象结构**：采用四叉树（quadtree），叶子节点定义superstate，支持膨胀（Expd）和折叠（Cllps）操作。
  - **多时间尺度学习**：三个时间尺度分别对应（1）快尺度：聚合演员-评论家（aggregated actor-critic）更新策略和值函数；（2）中尺度：Q-estimate更新用于指导树结构变化；（3）慢尺度：树结构（膨胀/折叠）更新。步长满足`lim η_t/ζ_t=0, lim ζ_t/ξ_t=0`。
  - **膨胀/折叠准则**：基于Q估计的差异`ˆΔ̄ϵ_{T,γ} = max_{γ'∈children} max_a ̄Q^{est}(γ',a) - max_a ̄Q^{est}(γ,a)`，若`ˆΔ̄ϵ ≥ 3(1-β)λ`则膨胀，反之折叠。λ是惩罚抽象大小的正则项。
  - **算法流程**：每步采样后更新聚合AC和几个Q估计（当前叶、展开子叶、折叠父节点），每若干步执行一次树结构更新。

## 6. 论文的主要结论与发现
- 性能驱动的抽象可以显著压缩状态空间（如16×16网格仅用34~55个superstate，压缩比>80%），同时保持接近最优的决策质量。
- 学习到的抽象在目标迁移时可通过热启动快速重规划，大大减少训练回合。
- 初始树深度存在U形效应：太深的树引入高维决策空间降低学习速度，太浅的树缺乏分辨率导致反馈不足；中等深度表现最佳。
- 抽象大小与性能之间存在帕累托边界：允许适度增加抽象大小可带来显著性能提升，但过度细化回报递减。
- 所提Tree-AC方法在所有任务上均优于扁平actor-critic和量化方法，且在大多数情况下优于CAT，尤其在大型环境（128×128）和Mountain Car中优势明显。
