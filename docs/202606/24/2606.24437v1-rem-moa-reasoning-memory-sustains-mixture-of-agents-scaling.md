## 摘要
混合智能体（MoA）架构通过将多个大语言模型（LLM）智能体组织成分层推理流水线，提升了推理时的扩展能力。然而，现有的MoA变体在深度增加时无法维持增益，出现性能退化、早期平台期或饱和现象。我们提出ReM-MoA，一种记忆增强的MoA框架，通过两种机制维持规模扩展：（1）排序推理记忆——使用比较性评审智能体持续存储并排序所有层的推理轨迹；（2）精选多样化记忆路由方案——将成功的和失败的轨迹的不同组合暴露给不同智能体，在传播高质量推理的同时保持探索多样性。我们进一步引入可选的多领域评审蒸馏流水线，通过前沿模型监督提升排序质量。在涵盖数学、形式逻辑、代码、知识和常识的五项推理基准测试中，ReM-MoA在深度和宽度扩展上均持续优于先前的MoA变体，且其优势随深度增加而扩大，确立了结构化跨层推理记忆作为可扩展多智能体推理的关键缺失机制。

## Abstract
Mixture-of-Agents (MoA) architectures improve inference-time scaling by organizing multiple LLM agents into layered reasoning pipelines. However, existing MoA variants fail to sustain gains as depth increases, exhibiting degradation, early plateauing, or saturation. We propose ReM-MoA, a memory-augmented MoA framework that sustains scaling through two mechanisms: (1) a Ranked Reasoning Memory that persistently stores and ranks reasoning traces from all layers using a comparative Reviewer Agent, and (2) a Curated Diversified Memory Routing scheme that exposes different agents to distinct combinations of successful and failed traces, preserving exploration diversity while propagating high-quality reasoning. We further introduce an optional multi-domain Reviewer distillation pipeline that improves ranking quality through frontier-model supervision. Across five reasoning benchmarks spanning math, formal logic, code, knowledge, and commonsense, ReM-MoA consistently outperforms prior MoA variants across both depth and width scaling, and its advantage widens with depth, establishing structured cross-layer reasoning memory as a key missing mechanism for scalable multi-agent inference.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：现有 Mixture-of-Agents (MoA) 架构在大规模扩展时（特别是增加层数深度）无法持续提升性能，反而出现三种失败模式——退化（degradation）、早期平台期（plateauing）或饱和（saturation）。
- **研究动机**：推理时计算已成为参数规模之外的另一条扩展轴，而多智能体协作（如 MoA）通过分层迭代有望实现可扩展的集体推理。然而，现有的 MoA 变体（如 Standard MoA、RMoA、AttentionMoA）均无法在深度增加时维持增益，这是因为：(1) 原始层间拼接会传播错误；(2) 仅局部通信无法保留早期高质量推理；(3) 智能体输入趋同导致探索多样性丧失。
- **整体含义**：该论文提出结构化跨层推理记忆是支撑可扩展多智能体推理的关键缺失机制，并以此设计了 ReM-MoA 框架。

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：将 MoA 扩展问题视为“质量-多样性联合维持”问题，而不仅仅是聚合或交互问题。通过建立跨层推理记忆，让后续层可以访问经过排序的早期推理轨迹，同时通过不同路由组合保持各层内智能体的探索多样性。
- **关键技术细节**：
  - **Ranked Reasoning Memory（排序推理记忆）**：每个层由 N 个提议者（proposer agent）产生推理轨迹 `r_l`，然后一个比较性评审智能体（Reviewer Agent）对所有轨迹进行评分（1-10整数）并生成针对性的理由（rationale），然后将 (轨迹, 分数, 理由) 三元组追加到记忆 `M` 中。记忆是只增的，后续层可以读取所有先前层的条目。
  - **Curated Diversified Memory Routing（精选多样化记忆路由）**：从第2层开始，每个提议者的提示中除了原始问题 `x` 外，还包含一个从记忆中选取的参考集 `S_{l,j}`。路由函数分为三类：对比式（Top a ∪ Bot b）、正增强（Top a）、失败模式聚焦（Bot a）。不同智能体获得不同的路由组合，从而保持多样性。例如 N=3 时，一个智能体获得 Top1∪Bot1，一个获得 Top2，一个获得 Bot2。
  - **可选增强：评审蒸馏（Reviewer Distillation）**：利用前沿大模型（如 GPT-5.5）作为教师，对同一问题产生的轨迹进行评分和理由标注，然后用 LoRA 微调一个小型评审模型（基座为 Qwen2.5-7B-Instruct），使评审质量提升，且能跨领域泛化。
- **算法流程（文字说明）**：
  1. 输入问题 `x`。
  2. 第1层：N 个提议者各自生成轨迹；评审智能体评分并写入记忆。
  3. 对于第 l ≥ 2 层：从记忆中为每个提议者分配不同的参考集（包含高分和低分轨迹及评审意见）；提议者生成新轨迹；评审智能体再次评分并写入记忆。
  4. 最后一层 L 后，聚合器（aggregator）也通过类似路由从记忆中获取参考，综合生成最终答案。
