## 摘要
零样本物体目标导航（ZS-OGN）要求具身智能体在没有任何先验训练的情况下探索并定位目标物体。为此，近期方法利用了基础模型，但它们通常依赖静态先验知识且缺乏适应性，导致重复错误和代价高昂的试错过程。本文提出一种自演化的零样本物体目标导航框架，能够在测试阶段持续改进。具体而言，我们通过从历史轨迹中提取可执行知识来构建一个代理规则记忆。然后，我们提出一种基于上置信界的检索策略，通过平衡语义相关性和历史成功次数来选择有效规则。此外，我们引入了一个记忆引导的预反射模块，在行动前预测潜在结果，从而减少低效探索。大量实验表明，我们的方法优于现有的零样本基线，在减少不必要步骤的同时，成功率提升了10.1%。

## Abstract
Zero-Shot Object-Goal Navigation (ZS-OGN) requires embodied agents to explore and locate target objects without any prior training. To this end, recent methods leverage foundation models. But they typically rely on static priors and lack adaptation, which leads to repeated errors and costly trial and error. In this paper, we propose a self-evolving ZS-OGN framework that enables continuous test-time improvement. Specifically, we build an agentic rule memory by extracting actionable knowledge from past trajectories. Then, we propose a retrieval strategy based on upper confidence bound, selecting effective rules by balancing semantic relevance and historical success. In addition, we introduce a memory-guided preflection module that forecasts potential outcomes before action, reducing inefficient exploration. Extensive experiments show that our method outperforms existing zero-shot baselines, achieving a 10.1\% improvement in success rate with fewer unnecessary steps.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：零样本物体目标导航（ZS-OGN）要求智能体在无任何训练的条件下，利用基础模型（如 LLM/VLM）在未知环境中探索并定位指定目标物体。
- **现有方法的局限**：当前方法依赖静态先验知识，无法根据实时反馈进行在线适应，导致反复犯相同错误（例如被视觉相似物体误导），且试错成本高（在有限步数预算下，每个动作都昂贵）。
- **研究目标**：提出一种能够实现**在线自适应**和**行动前预测**的零样本导航框架，在有限测试步数内减少冗余探索并提升成功率。

## 2. 方法论：核心思想、关键技术细节

### 核心思想：
将导航建模为一个**连续自我改进过程**：智能体通过从历史轨迹中提取可执行的导航规则，并利用这些规则在行动前预测失败风险，从而实现主动规避低效探索。

### 关键技术细节：

#### 2.1 自演化代理规则记忆（Inter-Episode Rule Self-Evolution）
- **规则生成**：每个导航回合结束后，LLM 分析完整轨迹，识别关键决策步骤，并从这些步骤的局部观测中提取通用导航规则，存入规则库 M。
- **语义驱动的信用分配**：对于轨迹中每一步计算语义相似度得分 S_i = α·S_room(sim) + (1-α)·S_obj(sim)，然后定义该步的信用 C_i = S_i - S_{i-1}。通过 softmax 归一化得到权重 w_i，新规则的支持分数 S_r = Σ w_i·C_i。
- **基于 UCB 的动态记忆管理**：将规则检索建模为多臂赌博机问题，计算每条规则的 UCB 得分：UCB(r) = μ_r + β·√(ln T / N_r)，其中 μ_r 是期望效用（通过动量更新：μ_r ← η·μ_r + (1-η)·S_r），T 是总回合数，N_r 是历史检索次数。新规则初始 UCB 设为无穷大，保证探索。最终选取 Top-K 规则注入 LLM 上下文。

#### 2.2 记忆引导预反射模块（Intra-Episode Preflective Exploration）
- **工作原理**：在每个决策时刻，智能体构建 2D 占用网格图并提取候选前沿。在行动前，检索规则库中的 Top-K 规则，让 LLM 先对所有候选前沿进行**失败风险预测**（结合当前观测和历史规则），然后选择风险最低、最有希望的前沿作为目标。
- **作用**：将被动的事后纠错转变为主动的风险规避，避免执行后才发现错误导致的步数浪费。

### 公式/算法流程（文字说明）：
1. 初始化规则库 M 为空。
2. 每个回合：
   - 空间映射 → 提取候选前沿。
   - 预反射：检索 Top-K 规则 → LLM 评估各前沿风险 → 选择最佳前沿。
   - 低层控制器导航至目标前沿（最多 H 步），若找到目标则 STOP。
3. 回合结束后：
   - LLM 分析轨迹，识别关键步骤。
   - 计算信用得分 S_r → 生成新规则。
   - 更新所有规则的 UCB 分数（动量平滑 + 频率计数），排序规则库。

## 6. 主要结论与发现

- **性能提升显著**：在 HM3D 上 SR 达 67.3%（比最强零样本基线 ASCENT 高 1.9%），在 MP3D 上 SR 达 49.0%（比 ASCENT 高 4.5%），同时 SPL 也有提升。
- **预反射和自演化记忆均有效**：单独使用预反射（+记忆）比基线在 HM3D 上提升约 0.8%（SR），在 MP3D 上提升 3.5%（SR）；单独使用自演化记忆提升更明显（MP3D 上 +4.5% SR）。
- **对语言模型骨干不敏感**：使用不同版本 Qwen（7B~9B）性能波动极小，说明框架主要受益于规则记忆和预反射机制，而非 LLM 规模。
- **定性结果展示更高效路径**：相比于基线方法，EvolveNav 的轨迹更直接，避免陷入死角和重复探索。

## 7. 优点

- **创新性**：首次将自演化记忆（UCB 管理的规则库）与行动前预反射相结合，解决零样本导航中缺乏在线自适应和试错成本高的问题。
- **实用性**：完全零样本、无需微调，所有模块在推理阶段运行，易于部署。
- **鲁棒性**：视觉模型固定为 BLIP-2，LLM 仅需 7B~9B 参数，性能稳定，资源需求相对可控。
- **实验设计严谨**：在两个主流基准上对比大量方法，消融实验清晰验证了各模块贡献。
