## 摘要
我们提出了一种基于强化学习的随机不确定性多智能体系统最优分布式控制算法。与现有方法不同，在优化的反步设计过程中，我们使用了行为者-评价者-辨识器结构。行为者神经网络用于反映控制行为，评价者神经网络用于评估控制性能，辨识器神经网络处理未知随机不确定性。此外，低通滤波器有效抑制了非仿射非线性故障带来的问题，并提出了一种混合事件触发控制策略以降低控制频率。我们分析了算法的运行，并提供了基于李雅普诺夫的稳定性证明，保证所有误差有界，确保领导者与跟随者之间的精确跟踪。我们在单轴机器人机械臂仿真中验证了其正确性，最后与非最优控制算法进行了比较，突出了我们最优控制算法的运行优势。

## Abstract
We propose a reinforcement learning (RL) based optimal distributed control algorithm for the multi-agent systems (MASs) with stochastic uncertainties. Unlike existing methods, during the optimized backstepping design process, we use the actor-critic-identifier structure. The actor neural network is used to reflect control behavior, the critic neural network works to evaluate control performance and the unknown stochastic uncertainties are handled by identifier neural network. Furthermore, a low-pass filter effectively suppresses problems stemming from non-affine nonlinear faults and a hybrid event-triggered control (ETC) strategy is proposed to reduce control frequency. We analyze our algorithm's operation, and we provide a Lyapunov-based stability proof that guarantees all errors are bounded, ensuring precise tracking between the leader and followers. We validate its correctness in a single-axis robotic manipulator simulation and finally, we compare against the non-optimal control algorithm highlighting our optimal control algorithm's operational advantages.

### 2. 论文提出的方法论

#### 核心思想
- 采用**反步设计**将高阶系统分解为n个子系统，对每个子系统的虚拟控制器进行RL优化。
- 利用**行为者-评价者-辨识器**神经网络结构：
  - **行为者神经网络**（Actor NN）：生成最优虚拟/实际控制输入。
  - **评价者神经网络**（Critic NN）：评估控制策略的长期代价。
  - **辨识器神经网络**（Identifier NN）：在线估计未知的随机动态函数和故障项。
- 设计**混合事件触发控制（ETC）策略**：
  - 当控制信号|u_i|低于阈值H_g时使用**静态触发条件**（固定阈值λ_i）。
  - 当|u_i|≥H_g时使用**动态触发条件**（阈值与|u_i|成正比加偏置λ*_i）。
  - 触发时刻更新控制器κ_i(t)，并利用tanh函数处理符号不连续带来的稳定性问题。
- 引入**低通滤波器**抑制非仿射非线性故障带来的高频振荡。

#### 关键技术细节
- **模型**：每个跟随者动态为 Ito 型随机微分方程（含 Wiener 过程 dw_i）：
  \[
  dx_{i,k} = (x_{i,k+1}+h_{i,k}(\bar{x}_{i,k}))dt + f_{i,k}(\bar{x}_{i,k})dw_i
  \]
  \[
  dx_{i,n} = (u_i + h_{i,n} + c_i\xi_i)dt + f_{i,n}dw_i
  \]
  其中c_i(t-t^*)描述故障的时变特性，c_i=0（t<t^*），c_i=1-e^{-μ(t-T0)}（t≥t^*）。ξ_i为非特定故障不确定性。

- **分布式一致性误差定义**：
  \[
  z_{i,1} = \sum_{j} p_{ij}(y_i-y_j) + q_i(y_i-y_r)
  \]
  \[
  z_{i,k} = x_{i,k} - u_{i,k-1}^* \quad (k=2,...,n)
  \]

- **RL优化过程**：
  - 对每个子系统的状态z_{i,k}定义性能指标J_{i,k} = ∫(z_{i,k}^2 + ū_{i,k}^2)dt。
  - 通过求解Hamilton-Jacobi-Bellman（HJB）方程得到最优虚拟控制器表达式：\( u_{i,k}^* = -\frac{\eta_i}{2} \frac{dJ_{i,1}^*}{dz_{i,1}} \)。
  - 由于非线性使得解析解不可得，利用NN近似，并设计行为者、评价者和辨识器的更新律：
    - 评价者NN：\( \dot{\hat{W}}_{ci,k} = -\phi_{ci,k} Q_{Ji,k}^T Q_{Ji,k} \hat{W}_{ci,k} \)
    - 行为者NN：\( \dot{\hat{W}}_{ui,k} = -Q_{Ji,k}^T Q_{Ji,k} (\phi_{ui,k}(\hat{W}_{ui,k}-\hat{W}_{ci,k}) + \phi_{ci,k}\hat{W}_{ci,k}) \)
    - 辨识器NN：\( \dot{\hat{W}}_{hi,k} = \delta_{i,k}(Q_{hi,k}z_{i,k}^3 - \sigma_{i,k}\hat{W}_{hi,k}) \)

- **混合ETC更新律**（以步骤n为例）：
  - 当|u_i|<H_g时：κ_i(t)=u_i^* - \bar{λ}_i \tanh(\bar{λ}_i z_{i,n}/Υ_i)，触发条件|κ_i - u_i| ≥ λ_i。
  - 当|u_i|≥H_g时：κ_i(t)=-(1+ϱ_i)(u_i^* \tanh(u_i^* z_{i,n}/Υ_i) + \bar{ϱ}_i \tanh(\bar{ϱ}_i z_{i,n}/Υ_i))，触发条件|κ_i - u_i| ≥ ϱ_i|u_i| + λ_i^*。
  - 最终控制信号u_i(t)=κ_i(t_s)，t∈[t_s,t_{s+1})。

- **稳定性证明**：构造Lyapunov函数V = Σ_i Σ_k (z_{i,k}^4/4 + NN权重误差项)，利用Young不等式、Lemma 1（tanh不等式）、Lemma 2（Hurwitz矩阵性质）和Lemma 3（SGUUB判定），推导出 \(\dot{V} \leq -\wp V + \Delta\)，从而所有误差信号半全局一致最终有界（SGUUB），同时证明最小执行间隔下界大于零（Zeno行为避免）。

### 7. 优点

- **统一处理多种不确定性**：同时处理随机不确定性（Wiener过程）和非特定故障（时变非仿射故障），实用性强。

- **混合ETC设计巧妙**：根据控制量幅值切换静态和动态触发条件，兼顾大信号下的节能与小信号下的精细调节，且通过tanh函数保证稳定性分析可行。

- **可扩展性**：框架基于反步，可推广到高阶系统；NN结构可处理未知动态。
