## 摘要
对于记忆增强的LLM智能体，记忆保留在何时至关重要？我们通过TraceRetain对此进行研究，这是一个轻量级框架，用于冻结的LLM智能体中的有界外部记忆，它根据可解释的特征（成功、年龄、访问频率、冗余度、特异性、相似性、下游效用）对条目进行评分，并在容量达到上限时驱逐得分最低的条目。在干净的ALFWorld环境中使用gpt-5-mini，外部记忆在两个随机种子下均显著优于无记忆方案，但在有界保留策略之间的差异落在Wilson 95%置信区间内：干净的ALFWorld在T=100到T=200时自然并未表现出记忆污染，而保留策略正是为解决此问题而设计的。在受控的噪声写入压力测试（75%的合成干扰项）下，无界记忆和FIFO-K50在Precision@5上分别下降（从20.2%降至12.4%，从15.8%降至3.8%），而TraceRetain-CEM基本保持不变（从16.9%降至16.6%），并保持了97/100的任务成功率。其机制是：无界记忆具有最高的平均相似度（0.87），但精度最低，表明嵌入空间中存在失败的干扰项接近查询。在留出分布内评估中，记忆增强策略在50个任务中解决了47到49个，而无记忆策略仅解决39个。在饱和的干净基准上，有界保留以零任务成功代价换取了记忆和步骤效率的提升，并且仅在输入流包含噪声时才与缓存启发式方法区分开来。

## Abstract
When does retention matter for memory-augmented LLM agents? We study this with TraceRetain, a lightweight framework for bounded external memory in frozen LLM agents that scores entries by interpretable features (success, age, access frequency, redundancy, specificity, similarity, downstream utility) and evicts the lowest-scoring ones at capacity. On clean ALFWorld with gpt-5-mini, external memory robustly improves over no memory across two seeds, but differences among bounded retention policies fall within Wilson 95% CIs: clean ALFWorld at T=100 to T=200 does not naturally exhibit the memory pollution retention is designed to address. Under a controlled noisy-write stress (75% synthetic distractors), unbounded memory and FIFO-K50 degrade on Precision@5 (20.2% to 12.4% and 15.8% to 3.8%) while TraceRetain-CEM is essentially unchanged (16.9% to 16.6%) and preserves 97/100 task success. The mechanism: unbounded memory has the highest mean similarity (0.87) but lowest precision, indicating failed distractors close to the query in embedding space. Held-out in-distribution evaluation shows memory-augmented policies solving 47 to 49 of 50 tasks vs. 39/50 for no memory. Bounded retention buys memory and step efficiency on saturated clean benchmarks at no task-success cost, and only differentiates from cache heuristics when streams contain noise.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：在长周期LLM智能体中，外部记忆的保留（retention）策略何时真正重要？即，在哪种条件下，记忆管理策略（尤其是选择性保留）能够带来实质性收益，而不仅仅是简单的缓存启发式方法（如FIFO、LRU）？
- **研究背景**：LLM智能体常依赖外部记忆来存储和检索过往经验，以实现非参数化适应。然而，随着记忆积累，可能出现“记忆污染”（memory pollution）——冗余、失败或无关的条目干扰检索质量，消耗prompt预算。现有研究多关注写入和检索，而对容量受限下的驱逐（eviction）策略研究不足。本文旨在通过实证方法明确有界保留策略的价值边界。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：设计一个轻量级的记忆保留框架TraceRetain，在冻结的LLM智能体中使用可解释特征对记忆条目评分，当记忆容量超过上限K时，驱逐得分最低的条目。该框架用作探针（probe），而非追求最优方法，目的是比较不同策略。
- **关键技术细节**：
  - **特征向量**：对每条记忆m_i在时间t计算特征φ(m_i, t)，包括：成功/失败、归一化年龄、最后访问间隔、log访问频率、特异性、冗余度、步骤效率、观察到的下游效用、效用计数置信度、最后检索相似度、平均检索相似度。
  - **评分函数**：s_i = w^T φ(m_i, t)，w为特征权重向量。两个变体：
    - TraceRetain-Linear：固定权重，手工设定（奖励成功、特异、频繁访问的记忆，惩罚旧、未使用、冗余或失败的记忆）。
    - TraceRetain-CEM：使用交叉熵方法（CEM）在20个任务的调优子集上搜索权重，优化目标为成功率、步骤效率和检索精度的组合。
  - **噪声写入压力**：为主动诱发记忆污染，每写入一条真实记忆后，紧接着写入三条同一任务但失败的干扰项，使得记忆库75%为合成噪声。
- **算法流程（文字说明）**：
  1. 智能体顺序解决一系列任务（episode t）。
  2. 每步嵌入当前任务，从记忆库M_t中检索最多k条记忆，使用冻结的ReAct-style LLM进行动作推理。
  3. 任务结束后，将轨迹摘要（任务、结果、交互历史）写入M_t。
  4. 若|M_t| > K，则计算每条记忆的评分s_i，驱逐得分最低的条目。

## 3. 实验设计：数据集、 benchmark、对比方法

- **数据集与场景**：
  - 主要benchmark：ALFWorld（文本化家庭操作任务），使用gpt-5-mini作为基础LLM，text-embedding-3-large作为嵌入模型。
  - 任务流：包含100或200个任务（T=100或T=200）。
  - 记忆容量K=50，检索top-5。
  - 五个条件：干净T=100（种子42和43）、干净T=200（种子42）、噪声写入T=100（种子42）、保留评估（在100个训练任务上写入记忆，然后在50个留出分布内任务上评估且不再写入）。
- **评估指标**：主要指标为任务成功率；辅助指标为Precision@5（检索的记忆中相关条目的比例，相关定义为成功且任务键匹配或同族同对象/容器）、平均环境步数。
- **对比方法**：

## 6. 论文的主要结论与发现

- **记忆增益普遍存在**：外部记忆在所有条件下均显著提升任务成功率（配对符号检验p<0.001），但无记忆基线在干净任务上也可达到86%-88%。
- **干净基准饱和，方法间差异不显著**：在干净ALFWorld（T=100或T=200）下，所有有界保留策略（包括简单FIFO）性能几乎相同，均处于Wilson 95% CI内。有界保留节省一半或更多内存，且不损失任务成功。
- **噪声写入下策略分化**：当75%写入为失败干扰项时，无界记忆和FIFO的Precision@5大幅下降（分别从20.2%降至12.4%、从15.8%降至3.8%），而TraceRetain-CEM基本不变（16.9%→16.6%），且只有TraceRetain变体的配对符号检验显著优于无记忆。
- **污染机制**：无界记忆的平均检索相似度最高（0.87）但精度最低，表明失败的干扰项在嵌入空间中离查询很近，导致检索污染；FIFO虽然相似度适中（0.81）但精度崩溃，因为插入顺序驱逐对污染视而不见。
- **留出迁移**：所有有界记忆方法在50个留出任务上均解决47-49个，而无记忆仅39个；TraceRetain-CEM达到最低平均步数（9.52）。
- **实用建议**：在干净饱和基准上，直接使用最简单的缓存启发式即可，无需负担学习策略；仅在确定存在噪声时，有界保留才体现出价值。

## 7. 优点

- **研究视角独特**：将记忆管理明确作为容量约束下的保留问题，与现有工作侧重写入和检索形成互补。
- **可解释特征设计**：使用11个可解释特征进行评分，便于理解和调试，且Linear变体无需额外LLM调用，轻量高效。
- **统计严谨性**：使用Wilson CI和配对符号检验，而不是简单的均值比较，增强了结论的可靠性。
- **噪声诊断实验**：有意构造记忆污染场景，揭示了污染的内在机制（高相似度低精度），并展示了不同策略的处理能力。
- **负结果的价值**：明确指出在干净条件下简单策略就足够，避免了过度宣传方法有效性，提供了实际部署的指导。
