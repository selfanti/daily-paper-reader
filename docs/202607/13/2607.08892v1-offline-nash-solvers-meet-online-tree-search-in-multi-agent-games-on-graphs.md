## 摘要
在多智能体追逃博弈中计算纳什均衡策略非常具有挑战性，因为联合状态和动作空间随智能体数量呈指数增长。现有方法要么依赖离线均衡近似（执行时可能缺乏适应性），要么依赖在线规划方法（存在巨大的分支因子）。本文提出原始引导树搜索（PGTS），一种融合离线精确纳什均衡计算与在线树搜索的混合框架：PGTS首先离线求解一组规模较小且易于处理的子博弈；部署时，PGTS在每个时间步执行在线树搜索，利用最优子博弈策略和价值函数引导树展开并估计叶节点价值。在不同图拓扑（包括真实网络）上的大量实验表明，PGTS显著优于当前最先进的学习和启发式基线，同时保持对对抗性策略的稳健性能。

## Abstract
Computing Nash equilibrium policies in multi-agent Pursuit-Evasion games (PEG) is challenging due to the exponential growth of the joint state and action spaces with the number of agents. Existing approaches either rely on offline equilibrium approximations, which may lack adaptability during execution, or online planning methods, which suffer from large branching factors. In this work, we propose Primitive-Guided Tree Search (PGTS), a hybrid framework that integrates offline exact Nash equilibrium computation with online tree search: PGTS first solves a collection of smaller, tractable sub-games offline; at deployment, PGTS performs online tree search at each time step, using the optimal sub-game policies and value functions to guide tree expansion and estimate leaf-node values. Extensive experiments on varied graph topologies, including real-world networks, demonstrate that PGTS significantly outperforms state-of-the-art learning and heuristic baselines, while maintaining robust performance against adversaries.

# 论文总结：Offline Nash Solvers Meet Online Tree Search in Multi-Agent Games on Graphs

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：多智能体追逃博弈（PEG）中，精确计算纳什均衡面临**联合状态空间与联合动作空间随智能体数量指数爆炸**的困境，导致传统方法（如Shapley值迭代）在智能体稍多时即不可行。
- **现有方法局限**：
  - **离线方法**（如PSRO、MT-PSRO）：训练后可快速执行，但训练复杂度随智能体数增长快，且训练时未见过的对手策略会导致性能严重下降。
  - **在线规划方法**（如SM-MCTS）：能根据当前状态实时推理，但分支因子随智能体数指数增长，搜索效率低下。
- **整体含义**：亟需一种兼顾离线的计算效率与在线的适应性的混合框架，在**可接受的计算成本**内实现**协调的多智能体决策**。

## 2. 方法论

### 核心思想
提出**原始引导树搜索（Primitive-Guided Tree Search, PGTS）**，将多智能体博弈分解为**若干较小规模的原始子团队博弈**（如1v1、2v1），离线精确求解这些子博弈的纳什均衡策略与值函数；在线执行时，在每个时间步运行树搜索，利用离线解引导扩展并估计叶节点价值。

### 关键技术细节

#### (1) 离线阶段：原始子团队博弈求解
- 选取两种典型原始游戏：**1 Red vs 1 Blue (1v1)** 和 **2 Red vs 1 Blue (2v1)**。
- 利用已有纳什均衡求解器（基于Shapley值迭代）在**完整状态空间**上精确求解，获得最优策略（π*_1v1, π*_2v1）和值函数（V*_1v1, V*_2v1）。
- 这些解被缓存，在线阶段调用。

#### (2) 在线树搜索（PGTS算法）
基于**同时移动MCTS（SM-MCTS）**框架，按标准四步迭代：

- **选择（Selection）**：使用两种选择算子之一：
  - **Regret Matching (RM)**：基于累积遗憾更新混合策略，可收敛到相关均衡。
  - **Decoupled UCT (DUCT)**：为每个队伍独立使用UCB（上置信界）准则，忽略联合动作耦合。
- **扩展（Expansion）**：**核心贡献之一——原始引导扩展**。
  - 对每个蓝色智能体，在其kr个最近红色智能体中，采样1v1和2v1原始策略生成的局部动作样本。
  - 将这些样本与启发式最短路径动作、随机动作一起构成**受限候选动作集**，从而**大幅降低分支因子**，同时保留战略性相关动作。
- **叶节点价值估计（Leaf Value Estimation）**：**另一核心贡献**。
  - 使用**分解框架**：先匹配2v1原始游戏（通过匈牙利算法），再对剩余代理匹配1v1原始游戏。
  - 叶节点价值 = 所有匹配的原始游戏值函数之和，近似最优分解的价值，避免指数级组合优化。
- **回溯（Backpropagation）**：将估计值沿访问路径向上传播，更新节点统计信息。

#### (3) 子团队分解基线（Section 3）
作为对比基线，给出一种直接使用原始游戏分解的策略：每个时间步，求解一个最大价值匹配（先2v1后1v1），然后各子团队独立按原始策略执行动作。

## 3. 实验设计

### 场景与环境
- 使用开源平台 **GraphChase**（提供图上的多智能体追逃基准）。
- 四种图拓扑：
  - **Grid 1 (Easy)**：7×7网格，简单结构。
  - **Grid 2 (Hard)**：7×7网格，更难布局。
  - **Scotland Yard**：200节点（来自著名桌游地图）。
  - **Atlanta**：151节点的真实世界道路网络。
- 实验聚焦**N个追捕者 vs 1个逃逸者 (Nv1)** 场景，以便与现有基准直接对比。

### 评估指标
- **最坏情况效用 (WCU)**：追捕者对逃逸者所有可行轨迹（到出口）的min对抗结果。
- **最短路径最坏情况效用 (SP-WCU)**：仅对逃逸者最短路径轨迹评估，用于对齐训练时的基线。
- **期望奖励 (ER)**：在随机采样逃逸者轨迹下的平均奖励。

### 对比方法
- **MT-PSRO**：基于策略空间响应或acles的多任务预训练方法（SOTA）。
- **NSGZero**：基于神经网络的MCTS（专为网络安全游戏设计）。
- **Decomposition**：本文提出的直接子团队分解基线。
- **Intercepting**：贪心最短路径启发式策略。

### 实验结果概要
- **PGTS-RM** 和 **PGTS-DUCT** 在所有图拓扑上均**一致优于所有基线**，尤其在复杂图（Scotland Yard、Atlanta）上WCU大幅领先。
- 在线搜索的**适应性**使其在面对非训练分布（如PGTS本身作为逃逸者）时仍能保持高奖励，而MT-PSRO、NSGZero则因过拟合而表现不佳。
- **消融实验**（Table 3）验证了：
  - 移除原始引导扩展（替换为启发式）后性能下降。
  - 仅使用1v1原始（无2v1）后性能显著下降，尤其在长时域图上。

## 6. 主要结论与发现

1. **混合框架有效**：PGTS将离线精确均衡与在线树搜索结合，同时实现了**低分支因子**和**长时域推理**，比纯离线方法（Decomposition）和纯在线方法（启发式MCTS）均显著优越。
2. **原始引导扩展是关键**：利用离线策略聚焦搜索空间，可在有限模拟预算下探索战略性重要区域。
3. **高阶原始（2v1）不可或缺**：仅在1v1原始下，无法捕获多智能体协调（如夹击），在复杂图上性能急剧下降。
4. **在线搜索赋予鲁棒性**：PGTS面对新对手（如PGTS逃逸者）时仍能保持高奖励，而学习型基线（MT-PSRO、NSGZero）因过拟合而失败。
5. **实用性**：离线计算一次后，在线决策时间约0.5秒/步，可适用于近实时场景。
