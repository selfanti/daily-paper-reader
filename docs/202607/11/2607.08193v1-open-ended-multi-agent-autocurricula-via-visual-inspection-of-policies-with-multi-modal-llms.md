## 摘要
强化学习中的开放式课程旨在通过识别有助于学习日益复杂技能的任务来训练具有通用能力的智能体。设计此类课程时的一个主要挑战是评估相对于智能体当前学习进度的任务难度。虽然先前的工作已经探索了使用标量任务分数或智能体行为的文本摘要，但在此我们研究一种不同的方法：通过记录的剧集视频直接检查策略行为。我们引入了该方法的一种简单而有效的实例化，利用视频语言模型（VLM）来处理这些视频并提供课程建议，我们将其称为策略视觉检查（VIP）。由于视频自然可以包含任意数量的可控智能体，我们在星际争霸多智能体挑战（SMAC）上对VIP进行了实证研究。我们表明，即使使用轻量级且可公开访问的VLM（VideoLLaMa2-7B），VIP也可以利用策略视频生成比纯文本消融和依赖于标量任务分数的方法更有效的课程。

## Abstract
Open-ended curricula in Reinforcement Learning (RL) aim to train generally-capable agents by identifying tasks that facilitate learning increasingly complex skills. A major challenge when designing such curricula is assessing task difficulty relative to the agent's current learning progress. While previous work has explored using scalar task scores or textual summaries of the agent's behavior, here we study a different approach: directly inspecting policy behavior via recorded episode videos. We introduce a simple yet effective instantiation of this approach which leverages a Video Language Model (VLM) to both process these videos and provide curriculum recommendations, which we call Visual Inspection of Policies (VIP). Since videos can naturally contain any number of controllable agents, we empirically study VIP on the StarCraft Multi-Agent Challenge (SMAC). We show that even with a lightweight and openly accessible VLM (VideoLLaMa2-7B), VIP can use policy videos to generate more effective curricula than both its text-only ablation and methods that rely on scalar task scores.

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

**核心思想**：利用视频语言模型（VLM）直接分析当前策略的剧集视频，并结合少量文本摘要（如胜率、历史任务列表），输出下一个“有趣且可解的”任务推荐。视频能自然包含多智能体行为的完整信息，且计算开销可控。

**关键技术细节**：
- **VLM 选择**：使用轻量级、可公开访问的 VideoLLaMa2-7B（7B 参数量化版本）作为 VLM，搭配句子相似度模型（all-MiniLM-L6-v2）进行输出消毒（sanitization）。
- **提示设计**：系统提示包含环境描述、可用的地图与难度列表、任务推荐规则（如避免重复、平衡难度）；任务提示包含当前任务的视频、胜率、历史任务列表。
- **输出消毒**：VLM 输出的原始任务推荐可能与可用任务不精确匹配（如幻觉、格式错误），通过计算 VLM 输出与候选任务名称的句子相似度，返回匹配度最高的有效任务；若低于阈值则回退到随机任务（实际未触发）。
- **任务空间**：SMAC 中任务由“地图”（决定敌我兵种与数量）和“难度”（1-10）组成，是一个有限可枚举集。

**算法流程**（伪代码）：
```
1. 初始化智能体策略 π⁰，初始任务 θ₀
2. 循环直到停止：
   a. 使用 MAPPO 算法在当前任务 θ_t 上训练策略 π_t (10^7 环境步)
   b. 评估当前策略：收集一个剧集视频 ϕ_video 和文本摘要 ϕ_text（胜率）
   c. VLM 接收 ϕ_video 和 ϕ_text，输出原始任务推荐 ϑ_t
   d. 消毒模块：将 ϑ_t 与可用任务集比较，得到有效的下一个任务 θ_{t+1}
   e. t = t+1
```
**核心特征**：VIP 不引入额外超参数（除 VLM 和 RL 算法所需），与智能体数量无关（只要所有智能体渲染在同一视频中），且仅需环境提供渲染功能（大多数模拟/真实 RL 域均满足）。

## 3. 实验设计：数据集/场景、Benchmark、对比方法

**环境**：StarCraft Multi-Agent Challenge (SMAC) v1，合作式多智能体对战环境。任务由地图（编队组合）和难度（1-10）定义。

**数据划分**：
- 训练地图：12 个（80%），测试地图：3 个（20%）：10m vs 11m、3s vs 4z、3s5z（均为难度 5）。
- 这些测试地图在所有课程训练中从未出现，用于评估泛化能力。

**Benchmark 设置**：
- 每个课程包含 16 个任务（每个任务训练 10^7 环境步）。
- 最终评估：对测试任务进行 10^7 步微调，记录胜率（95% 置信区间，5 个随机种子）。

**对比方法**：
- **VIP（本文）**：使用 VLM + 视频。
- **Text-Only 消融**：除视频输入及提及视频的提示被移除外，与 VIP 完全相同。
- **PLR-PVL**：Robust Prioritized Level Replay 使用 Positive Value Loss 分数。
- **PLR-MaxMC**：同上但使用 Maximum Monte Carlo 分数。
- **Random**：随机选择任务。
- **MAPPO 基线**：仅 MAPPO 从头训练（无课程）。
- **MAPPO\*（监督搜索）**：使用 Yu et al. (2022) 通过网格搜索找到的最佳超参数（每任务约 1728 种组合），用于与 VIP 进行非对称比较。

**额外评估**：还对所有课程中频繁出现的 3 个任务（3m 难度 6、3s vs 3z 难度 7、3s vs 5z 难度 7）进行了微调测试，以衡量知识保留能力。

## 6. 论文的主要结论与发现

1. **视频信息关键性**：VIP 在 3s vs 4z 和 3s5z 上分别达到 84% 和 76% 胜率，而 Text-Only 消融几乎为 0%，证明视频信息本质性地提升了课程质量。
2. **超越标量分数方法**：VIP 在所有测试任务上显著优于 PLR-PVL 和 PLR-MaxMC，后两者几乎无法学出有效策略（胜率接近 0%）。
3. **保留知识能力强**：在频繁出现于课程中的任务上（如 3s vs 3z），VIP 微调后实现 100% 胜率，且收敛更快。
4. **计算效率高**：采用轻量 VLM，VLM 推理仅占约 1% 总计算时间，不构成瓶颈。
5. **与监督方法的比较**：VIP 使用不到 1/100 的环境交互量（1.6e8 vs 1.7e10），在 3s5z 上取得与 MAPPO\* 相当的胜率（76%），证明了无监督课程的价值。
