## 摘要
最近统一的多模态模型表明，单一架构可以同时执行视觉/语言理解和图像生成/编辑。然而，它们反复将所有历史视觉和文本输入馈送到共享的上下文窗口中，由于视觉标记爆炸和不可靠的跨轮引用，限制了长程多模态对话。我们提出了一种认知结构的多模态智能体，该智能体将视觉信息外化到情景视觉记忆中，并在推理过程中选择性地重新激活相关情景。该智能体包括：用于结构化视觉抽象的感知抽象引擎、用于跨轮记忆检索的认知检索引擎，以及用于自主任务推理和动作规划的多模态执行控制器。为了解决现有数据集中缺乏轮级检索监督的问题，我们开发了一个统一情景引擎，该引擎通过程序生成具有细粒度检索注释的结构化多轮对话，使得强化学习能够优化抽象和检索策略。我们还构建了一个按难度分层的长程视觉对话基准，以评估情景视觉回忆。我们的8B参数智能体在20轮会话中实现了91.4%的检索准确率，比32B基线高出8.2%，同时每轮推理时间减少了近一半（从23.1秒降至12.7秒）。我们进一步提出了认知结构多模态智能体框架（CMA-Harness），这是同一认知结构的工具增强部署，集成了持久多模态记忆、网络访问、图像生成/编辑/合成工具以及兼容OpenAI的推理服务。与单一参数缩放相比，结构化记忆和模块化决策为长程多模态智能体提供了一种更具可扩展性和高效性的范式。代码：https://github.com/caseclose/cma-harness ; 项目页面：https://caseclose.github.io/cma-harness/

## Abstract
Recent unified multimodal models show a single architecture can jointly perform vision/language understanding and image generation/editing. However, they repeatedly feed all historical visual and textual inputs into a shared context window, limiting long-horizon multimodal dialogue due to visual token explosion and unreliable cross-turn referencing. We propose a Cognitive-structured Multimodal Agent that externalizes visual information into an Episodic Visual Memory and selectively reactivates relevant episodes during reasoning. The agent consists of a Perceptual Abstraction Engine for structured visual abstraction, a Cognitive Retrieval Engine for cross-turn memory retrieval, and a Multimodal Executive Controller for autonomous task inference and action planning. To address the lack of turn-level retrieval supervision in existing datasets, we develop a Unified Scenario Engine that programmatically generates structured multi-turn conversations with fine-grained retrieval annotations, enabling reinforcement learning to optimize abstraction and retrieval policies. We also construct a long-horizon visual-dialogue benchmark stratified by difficulty to evaluate episodic visual recall. Our 8B agent achieves 91.4% retrieval accuracy over 20-turn sessions, surpassing 32B baselines by +8.2% while nearly halving per-turn inference time (23.1s -> 12.7s). We further present the Cognitive-structured Multimodal Agent Harness (CMA-Harness), a tool-augmented deployment of the same cognitive structure integrating persistent multimodal memory, web access, image generation/editing/composition tools, and OpenAI-compatible serving. Structured memory and modular decision-making offer a more scalable, efficient paradigm for long-horizon multimodal agents than monolithic parameter scaling. ## 2. 论文提出的方法论

### 2.1 核心思想：认知结构多模态智能体（Cognitive-structured Multimodal Agent）

将单一大模型解耦为三个独立训练、可替换的模块：

| 模块 | 功能 |
|------|------|
| **感知抽象引擎（Perceptual Abstraction Engine, PAE）** | 将每张输入图像压缩为结构化记忆条目（描述性字幕、属性标签、压缩略缩图），存入外部 **情景视觉记忆（Episodic Visual Memory, EVM）**，不占用对话上下文窗口。 |
| **认知检索引擎（Cognitive Retrieval Engine, CoRE）** | 根据当前查询和对话状态，从 EVM 中检索最相关的视觉情景索引列表，支持多模态跨轮推理。 |
| **多模态执行控制器（Multimodal Executive Controller, MEC）** | 融合对话历史与检索到的记忆，自动推断任务意图（理解、生成、编辑、合成、纯聊天），并规划最终动作——调用图像生成/编辑模型或直接文本回复。 |

### 2.2 关键技术细节

- **优化目标**：不优化传统的字幕质量指标（CIDEr、BERTScore），而是直接以 **检索成功度** 作为奖励信号。通过奖励 PAE 和 CoRE 共同优化记忆构建与检索策略。
- **训练流程**：三个阶段：  
  1. **SFT on CoRE**：在有检索标注的数据上对检索引擎进行监督微调。  
  2. **RL on CoRE（DAPO）**：使用难度加权 Jaccard 相似度作为奖励，进一步优化检索策略。  
  3. **RL on PAE**：冻结 CoRE，检索准确率作为 PAE 的奖励，使记忆表示直接服务于下游检索任务。
- **奖励函数**：  
  \( r = \frac{|\hat{R} \cap R^*|}{|\hat{R} \cup R^*|} \cdot w_d \)，其中 \( w_d = 1.2 \) 用于困难/非常困难轮次，负样本正确空返回奖励 1，格式错误有惩罚。

### 2.3 训练数据生成：统一情景引擎（Unified Scenario Engine）

- 为解决现有数据集缺乏轮级检索标注的问题，该引擎以程序化方式生成 20 轮多模态对话，每轮自动产生：用户输入、任务类型、真实检索集合、难度标签、挑战标签。
- 包含 55 个主题、8 个领域，通过 Gemini 模拟用户并注入结构归纳偏置（如“至少 3 次跨度 ≥8 轮的回调”）。
- 训练数据：270 个会话（5,400 轮）用于 SFT，30 个会话用于验证，RL 对困难子集采样。

### 2.4 算法流程（文字描述）

1. 每轮用户输入后，MEC 先判断任务类型（生成/编辑/理解/聊天）。  
2. 若需视觉上下文，则调用 CoRE 基于当前查询和对话状态从 EVM 中检索相关记忆条目。  
3. 将检索到的历史图像、记忆元数据与对话历史拼接，形成紧凑上下文。  
4. 对于生成/编辑任务，MEC 重写提示词并调用图像生成/编辑模型；对于理解/聊天任务，直接生成文本响应。  
5. 每轮只输入检索到的少量历史图像，而不是全部，大幅减少视觉 token。

## 6. 论文的主要结论与发现

1. **统一模型在长程对话中检索能力极差**（BAGEL 在 Hard 子集准确率 <3%）。  
2. **结构化记忆过滤显著优于全上下文拼接**：Multi-Agent Baseline-8B 相比 All-Context-8B 在 Full 上提升 2.9%（英文），在 Medium 上提升 8.2%。  
3. **任务特定训练超越参数扩展**：8B 的 SFT+RL 模型在 Full 上比 32B Multi-Agent 基线高 8.2%，且差距随难度增大而扩大（Hard 上 +18.4%）。  
4. **效率优势**：推理时间从 23.1s 降至 12.7s（1.8× 加速），同时生成质量更高。  
5. **多阶段 SFT+RL 均有效**：SFT 提升 7.3%，RL on CoRE 再提升 6.5%，RL on PAE 再提升 4.6%。  
6. **文本型记忆不足以应对细粒度视觉召回**：本文 EVM（含略缩图）在 Hard 子集准确率 82.0%，远高于文本型（MemoryLLM: 24.4%, ReasoningBank: 39.0%）。

## 7. 优点

1. **创新框架**：将认知科学中的情景记忆与模块化架构引入多模态智能体，突破统一模型的上下文瓶颈。  
2. **数据生成方案**：通过统一情景引擎自动产生带细粒度检索标注的长程对话数据，解决监督不足。  
3. **强化学习闭环**：直接以检索成功度作为奖励，训练 PAE 和 CoRE 相互配合，优化目标与下游一致。
