## 摘要
训练工具使用智能体从自身经验中改进仍然具有挑战性，因为监督微调依赖于固定的教师蒸馏轨迹，而稀疏奖励强化学习为长程交互提供的监督较弱。我们提出了DeepSearch-Evolve，这是一个基于DeepSearch-World构建的网页智能体自蒸馏框架，DeepSearch-World是一个确定且可验证的环境，具有可复现的搜索和页面阅读工具。DeepSearch-World包含从实体级随机游走构建的42万个多跳问答任务，并支持对自我进化有用的关键智能体认知行为，包括进度验证、基于情境的反思和错误恢复。DeepSearch-Evolve迭代执行轨迹生成、过滤、数据混合和微调，以训练更强的智能体。无需从更强大的模型进行蒸馏，DeepSearch-World-9B在与开源智能体相比时取得了有竞争力的性能，在BrowseComp上达到31.2%，在GAIA上达到61.5%，在HotpotQA上达到93.4%，这表明可验证环境能够使长程网页智能体实现可扩展的自我进化。我们将发布环境、42万训练池、验证集、模型和代码，以促进未来关于自我改进深度搜索智能体的研究。

## Abstract
Training tool-use agents to improve from their own experience remains challenging, as supervised fine-tuning relies on fixed teacher-distilled trajectories, while sparse-reward reinforcement learning provides weak supervision for long-horizon interactions. We present DeepSearch-Evolve, a self-distillation framework for web agents built on DeepSearch-World, a deterministic and verifiable environment with reproducible search and page-reading tools. DeepSearch-World contains 420K multi-hop QA tasks constructed from entity-level random walks and supports key agentic cognitive behaviors useful for self-evolving, including progress verification, grounded reflection, and failure recovery. DeepSearch-Evolve iteratively performs trajectory generation, filtering, data mixing, and fine-tuning to train stronger agents. Without distillation from more capable models, DeepSearch-World-9B achieves competitive performance compared with open-source agents, reaching 31.2% on BrowseComp, 61.5% on GAIA, and 93.4% on HotpotQA, showing that verifiable environments enable scalable self-evolution for long-horizon web agents. We will release the environment, 420K training pool, validation set, model, and code to facilitate future research on self-improving deep search agents.

## 1. 核心问题与整体含义

- **研究动机**：训练工具使用（tool-use）智能体从自身经验中自主改进是当前大语言模型智能体研究的核心挑战。现有方法存在明显局限：
  - **监督微调（SFT）** 依赖固定教师蒸馏轨迹，性能易饱和于骨干模型能力上限。
  - **稀疏奖励强化学习（RL）** 仅提供轨迹级弱监督，无法区分失败来源（如查询构建、工具选择、证据提取或答案合成）。
- **整体含义**：本文主张，要实现长程（long-horizon）网络智能体的可扩展自进化，必须构建**确定性且可验证的工具使用环境**，使过程级（process-level）监督变得可靠。基于此，作者提出了DeepSearch-World环境与DeepSearch-Evolve自蒸馏框架，使9B参数模型不依赖更强模型蒸馏即可取得与开源智能体竞争的性能。

## 2. 方法论

### 核心思想
通过迭代式自蒸馏（self-distillation）循环，让智能体在可验证环境中生成自己的轨迹，过滤出成功轨迹并转换为标准ReAct格式进行SFT训练，从而逐步注入规划、记忆、反思、错误恢复等高级认知能力。

### 关键技术细节

#### 2.1 可验证工具环境 DeepSearch-World
- **数据构建**：基于Wikipedia超链接图，采样实体级随机游走（entity-level random walks），构造42万条多跳QA任务。将实体模糊化（如“2014”→“the 2010s”），迫使智能体进行真实搜索推理。
- **离线语料库**：爬取相关Wikipedia页面，构建约1000万条条目的本地语料库，包含标题、摘要和全文。
- **工具实现**：
  - `web_search_wiki`：基于Lucene BM25检索，返回标题、摘要、URL列表。
  - `visit_wiki`：通过SQLite偏移索引实现确定性页面读取。
  - 工具模式与真实网络搜索/浏览一致，可无缝替换为实网API。
- **环境级验证与反思**：存储每个问题的目标实体集合，每次工具调用后自动检查是否命中未解决实体，提供进度验证和分阶段规则反思（从泛泛提示到逐步具体化）。

#### 2.2 脚手架教师智能体（Scaffold Teacher）
分为三个阶段：
- **Plan**：初始化结构化进度状态（completed_list, todo_list, experience, information），进行任务分解。
- **Act**：执行最多 Tmax 步工具交互，每步根据环境反馈更新进度状态；环境提供基于规则的多级反思。
- **End**：证据充足时基于已验证的工作记忆生成简洁答案，减少幻觉。

#### 2.3 自进化训练循环 DeepSearch-Evolve
迭代步骤：轨迹生成 → 拒绝采样（答案正确） + 质量过滤（LLM评估冗余性、目标对齐、逻辑一致性） → Scaffold-to-ReAct转换（将状态更新和反思重写为<think>块中的自校正） → 重要性采样混合历史轨迹 → SFT更新（每个epoch一轮）。异步生成与训练，生成不阻塞训练。

- 更新可视为硬标签的token级策略蒸馏（公式(3)为NLL损失）。
- 引入重要性采样（decay γ=0.5）以优先近期数据同时缓解灾难性遗忘。

## 3. 实验设计

- **数据集与基准**：使用7个基准，涵盖深度搜索与推理：

- **训练配置**：

## 6. 主要结论与发现

1. **可验证环境使自进化可行**：DeepSearch-World提供的确定性观察、实体级验证和反思信号，使教师智能体能稳定生成高质量轨迹，学生智能体从中学习规划、记忆、校正等行为。
2. **无需更强教师蒸馏**：仅依赖自身环境验证的轨迹，9B模型在BrowseComp（31.2%）、GAIA（61.5%）、HotpotQA（93.4%）等基准上接近甚至超越需要更强模型蒸馏的开源模型。
3. **数据规模与多样性关键**：420K大池比100K小池提供更优的验证表现和更低的格式错误，避免过拟合于窄模式。
4. **组件有效性**：拒绝采样是最主要的增益来源；反射改写和状态内化均带来一致提升；Scaffold-to-ReAct转换有效注入高级认知技能。
5. **工具使用行为显著改善**：DeepSearch-World智能体平均交互轮数从骨干的4.7提升至18.0，访问调用从0.9增至5.4，高级能力评分从19%升至70%。

## 7. 优点

- **环境设计创新**：提供确定性、可复现、可验证的离线环境，支持过程级验证（实体命中检测），为潜在不稳定的自我进化提供坚实基础。
- **自蒸馏框架稳定高效**：采用SFT而非分布匹配的on-policy蒸馏，避免在低质量状态上获得噪声教师分布；异步生成+训练提升吞吐。
- **认知技能注入**：通过Scaffold-to-ReAct转换将显式规划、工作记忆、失败恢复注入标准ReAct策略，无需模型依赖外部提示。
