## 摘要
基于可验证奖励的无评论家强化学习（RLVR），以组相对策略优化（GRPO）为例，避免了训练价值函数（评论家），并相对于基于评论家的PPO流程减少了内存和计算开销，用于对齐大型语言模型。然而，GRPO风格的优势估计依赖于提示局部（提示组内）的奖励统计，可能不稳定。特别是，当提示组中的所有生成结果获得相同奖励时，组内奖励方差变为零，组归一化导致该组优势为零，从而在具有二元验证器的冷启动场景中阻碍学习。我们引入了BV-Blend，一种无评论家框架，通过将提示局部在策略统计与语义簇条件历史矩相结合来稳定优势估计。BV-Blend维护每个簇的EMA追踪奖励矩，从标准误（SEM）代理推导出置信权重，并利用该权重将历史与提示局部基线和方差统计混合，为PPO风格的裁剪更新生成标准化优势。在可验证推理基准上的实验表明，BV-Blend提高了训练稳定性和性能，并且在组归一化方法可能停滞的场景中保持稳健。

## Abstract
Critic-free reinforcement learning with verifiable rewards (RLVR), exemplified by Group Relative Policy Optimization (GRPO), avoids training a value function (critic) and reduces memory and compute overhead relative to critic-based PPO pipelines for aligning large language models. However, GRPO-style advantage estimation depends on prompt-local (within-prompt-group) reward statistics and can be unstable. In particular, when all rollouts in a prompt group receive identical rewards, the within-group reward variance becomes zero, and group normalization yields zero advantages for that group, impeding learning in cold-start regimes with binary verifiers. We introduce BV-Blend, a critic-free framework that stabilizes advantage estimation by combining prompt-local on-policy statistics with semantic-cluster-conditioned historical moments. BV-Blend maintains EMA-tracked reward moments for each cluster, derives a confidence weight from a standard error of the mean (SEM) proxy, and uses this weight to blend historical and prompt-local baseline and variance statistics into a standardized advantage for PPO-style clipped updates. Experiments on verifiable reasoning benchmarks show that BV-Blend improves training stability and performance, and remains robust in regimes where group-normalized methods may stall.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题背景**：基于可验证奖励的强化学习（RLVR）中，无评论家方法（如GRPO）通过组内奖励归一化计算优势，避免训练价值函数，降低计算开销。
- **核心瓶颈**：在冷启动或奖励信号稀疏（如二元验证器）时，同一提示组的所有生成结果可能获得相同奖励（全正或全负），导致组内方差为零，组归一化产生的优势为零，学习信号完全消失，阻碍策略更新。
- **研究意义**：需要一种无需评论家的稳定优势估计方法，在零方差场景下仍能提供有效学习信号，提升训练稳定性与最终性能。

## 2. 论文提出的方法论
### 核心思想
- 通过语义聚类维护历史奖励矩（EMA更新），基于历史估计的不确定性（标准误代理）计算置信权重，将历史与当前提示组内的基线和方差统计进行加权混合，得到单一标准化优势，用于PPO风格裁剪更新。

### 关键技术细节
1. **语义聚类**：使用冻结编码器将提示嵌入，并分配给固定K-means码本中的簇（离线训练，RL期间不变）。
2. **历史EMA矩**：为每个簇维护三个EMA量：均值 \(m_1(k)\)、二阶原始矩 \(m_2(k)\)、有效样本量 \(N_{\text{eff}}^k\)。更新率为 \(\gamma\)，冷启动时初始化。
3. **不确定性转置信权重**：
   - 标准误代理：\(\text{SEM}_{\text{hist}}(k) = \sigma_{\text{hist}}(k) / \sqrt{N_{\text{eff}}^k + \delta_N}\)
   - 置信权重：\(w_k = \exp(-\text{SEM}_{\text{hist}}(k) / T)\)，T控制敏感度。较低不确定性对应较大权重（更多依赖历史）。
4. **基线-方差混合与优势计算**：
   - 混合基线：\(b(m) = w(m)\mu_{\text{hist}}(k(m)) + (1-w(m))\mu_G^{(m)}\)
   - 混合方差：\(s^2(m) = w(m)v_{\text{hist}}(k(m)) + (1-w(m))(\sigma_G^{(m)})^2\)
   - 标准化优势：\(A_{\text{BV},i}^{(m)} = (R_i^{(m)} - b(m)) / (s(m) + \delta)\)
5. **更新顺序**：每个batch，先用更新前的历史矩计算\(w_k\)；优化策略后再更新EMA矩。不反向传播通过任何奖励统计量。

### 算法流程（文字说明）
- 对每个提示，采样G条轨迹，获得奖励。
- 计算组内均值与标准差。
- 将提示嵌入并分配簇。
- 若簇首次出现，置\(w=0\)（纯提示局部归一化）；否则从EMA读取历史矩，计算SEM和权重。
- 混合得到b和s，对每条轨迹标准化得到优势。
- 用PPO裁剪目标更新策略。
- 聚合本周期的簇统计，更新EMA矩。

## 6. 论文的主要结论与发现
- **有效性**：BV-Blend在域内数学推理上平均51.7%，领先所有对比方法（包括混合/离策略方法如LUFFY†的51.0%）；在OOD泛化上平均64.1%，同样最优。
- **稳定性**：训练中有效信号比（非零归一化尺度的提示比例）显著高于GRPO，避免优势消失；策略熵保持更高，探索更持久。
- **鲁棒性**：在分数较低或分布偏移的骨干模型（如Llama-3.1-8B）上，BV-Blend仍表现出改善，而GRPO等可能退化。
- **消融结论**：单独使用有效计数或历史尺度均有提升，但合并为SEM权重效果最佳；朴素历史平均反而下降，说明选择性混合是关键。
