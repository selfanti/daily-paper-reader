## 摘要
强化学习已被证明是创建先进人工智能和机器人系统的宝贵工具，为从游戏对弈到机器人技术再到基础模型等各个领域做出了贡献。这些AI系统通常通过试错学习一种近乎最优的行为来解决其任务。然而，在许多用例中，人们希望在一定程度上实时控制任务的解决方式。我们将这些对核心任务的修改称为“风格”。我们将通用价值函数近似器（UVFAs）与精心选择的训练场景、学习算法和数据增强相结合，创建了一个在复杂领域中展示风格的教练代理框架。我们在AAA级电子游戏《地平线：西之绝境》和《GT赛车》中，以及一个开源的人形机器人测试领域中展示了该框架的应用。尽管这些领域性质不同——赛车、风格化游戏战斗和人形行走——每个代理在对风格请求表现出强一致性的同时，仍然满足了其领域中的主要任务。重要的是，本文概述的技术允许最终用户在运行时选择最终行为，从而对最终执行的表现提供灵活的控制。

## Abstract
Reinforcement learning has proven to be a valuable tool in the creation of advanced AI and robotic systems, contributing to everything from game playing to robotics to foundation models. Through trial-and-error, these AI systems typically learn one, near-optimal behavior to solve their tasks. However, there are many use cases in which one would like to assert some level of control, preferably in real time, over how the task is solved. We refer to these modifications of a core task as styles. We combine universal value function approximators (UVFAs) with carefully selected training scenarios, learning algorithms, and data augmentation to create a framework for coaching agents that exhibit styles in complex domains. We demonstrate the framework's application in the AAA video games Horizon Forbidden West and Gran Turismo, and in an open-source humanoid test domain. Despite the different nature of the domains -- car racing, stylized game combat, and humanoid walking -- each agent shows strong coherence to the style requests while still satisfying the main task in its domain. Importantly, the techniques outlined in this paper allow an end user to choose the final behavior at run time, giving them flexible control over the final executed performance.

## 一、论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：标准强化学习训练出的智能体通常只学习一种近最优的单一行为，缺乏在运行时实时调整任务执行方式的灵活性。然而，许多实际场景（如清洁机器人根据婴儿睡眠选择安静模式、玩家在游戏中切换战斗风格）要求用户能在不改变核心任务的前提下，动态控制行为偏好，即“风格”。
- **整体含义**：本文提出“可教练代理”（coachable agent）概念，将核心任务（如击败敌人、赛车竞速、人形行走）与风格（如武器偏好、轮胎节省、步态长度）解耦，使智能体在保持任务性能的同时，能根据运行时指令展现多种风格。该工作将通用价值函数近似器（UVFA）扩展到高维复杂控制域（AAA游戏和人形机器人），并引入一系列工程创新（场景训练、数据增强、分层回放缓冲等）实现可扩展训练。这项技术对游戏产业（QA、NPC增强、辅助功能）、机器人领域（用户定制行为）有直接应用价值。

## 二、方法论：核心思想、关键技术细节、公式/算法流程

### 核心思想
- 将奖励函数拆分为固定任务奖励 `r_task(s,a)` 和风格奖励 `r_style(s,a,θ)`，其中 θ 是多维风格参数向量（包括权重、阈值、目标值）。
- 使用**风格条件化通用价值函数近似器**（style-conditioned UVFA）：策略和评论家网络同时以状态 `s` 和风格参数编码 `f(θ)` 为输入，即 `π(a|s, f(θ))` 和 `Q(s, f(θ), a)`。一个策略可表达多种行为模式，无需重新训练。

### 关键技术细节
1. **风格参数采样**：在每回合开始时固定 θ 以避免时序信用分配问题。训练时从混合分布采样：3% 默认（全零），10% 各维度独立采样，87% 单热采样（一次仅激活一个风格）。
2. **奖励设计**：`r = r_task + Σ w_i * g_i(s, a; ξ_i)`，其中 `w_i` 为权重，`g_i` 为风格奖励函数（如弹药使用、部件移除、元素伤害等）。权重可正可负（鼓励或禁止风格）。
3. **算法选择**：
   - 人形域：SAC（软演员-评论家）
   - GT赛车：QR-SAC（分位数回归 SAC）
   - HFW（地平线）：新算法 **Cat-RAC**（Categorical Regularized Actor-Critic），是对 SAC 的扩展，支持混合动作空间（离散按钮+连续摇杆），使用分类批评家（255 个 bin）和 HL-Gauss 目标，以及 Gumbel-Softmax 实现分类策略梯度。
4. **训练框架**：
   - 异步 rollout workers 拉取最新策略，每回合采样 UVFA 权重。
   - 使用**场景训练**（scenario training）增强多样性。
   - **分层采样**：GT 使用多表（multi-tables），HFW 使用事件表（event tables）按风格配置组织回放。
   - **任务模仿**（task impersonation）：对每个采样到的转移，额外生成 n 个不同 θ 下的标签（n=4），扩展风格覆盖而不增加回放缓冲。
5. **具体域实现**：
   - HFW：混合动作空间（315 种离散组合+4 个连续值），策略每 6 帧执行一次。特征包括玩家/敌人状态、局部坐标编码等。
   - GT：使用连续动作，风格控制包括漂移、轮胎/燃油节省、侵略性（纵向/横向距离阈值）。
   - 人形：控制 21 维连续动作，风格包括手臂姿势（三个离散姿势）和步长（连续目标值）。

### 公式/算法流程（文字描述）
1. 初始化策略网络 π(a|s, f(θ)) 和批评家 Q(s, f(θ), a)。
2. 每回合从分布 pΘ 采样风格参数 θ。
3. rollout worker 与环境交互，记录转移 (s, a, r, s')，其中 r 包含 r_task 和 r_style(θ)。
4. 转移存入分层回放缓冲（多表/事件表）。
5. 训练时，从缓冲均匀采样 mini-batch，并执行任务模仿（对每个转移额外生成 n 个 θ' 下的奖励标签）。
6. 使用策略-批评家顺序更新（先策略后批评家）以提高稳定性。
7. 重复直到收敛。

## 三、实验设计：数据集/场景、基准、对比方法

2. **Gran Turismo 7 (GT7)**  
   - 场景：不同赛道（Trial Mountain, Lago Maggiore, Le Mans），不同赛车（Lamborghini Countach, Red Bull X2019 等）。

## 六、主要结论与发现
1. **风格一致性高**：HFW 中，请求的风格与智能体实际造成的伤害类型高度相关（图3a 对角线主导）。默认行为偏向 SkyKiller 武器，但“禁止 SkyKiller”风格成功抑制了使用。
2. **风格可组合**：HFW 和 Humanoid 域都展示了组合两种风格（如特定武器+特定元素）的可行性，且精度保持。
3. **性能权衡**：风格权重增加会提高风格得分但降低胜率（图4b），用户可通过调节权重在 Pareto 前沿上选择工作点。
4. **泛化能力**：在未见过的敌人和场景（OOD）中，智能体仍保持高胜率（图4a），仅双强敌场景略有下降。
5. **学习上下文化**：HFW 智能体不仅机械执行风格指令，还能根据敌人弱点和状态智能选择武器（如先使用冰冻弹药再切换到高伤害武器）。
6. **GT 中精确行为控制**：通过调节燃油/轮胎惩罚权重，可延长 pit stop 间隔60%，同时圈速损失 2-3 秒；通过调节侵略性距离阈值，可平滑控制超车倾向和碰撞率。
7. **人形中实现连续和离散风格组合**：手臂姿势和步长可同时控制，尽管手臂姿势影响重心平衡。

## 七、优点
- **方法创新**：将 UVFA 扩展到风格条件化，并针对高维域提出 Cat-RAC 算法（混合动作空间、分类批评家、Gumbel-Softmax + regret matching），解决离散动作梯度回传问题。
- **实用性强**：支持实时风格切换，用户无需重新训练即可调整行为，对游戏和机器人领域有实际推广价值。
- **跨域验证**：在三个截然不同的复杂领域（AAA 游戏战斗、赛车、人形行走）中均取得良好效果，证明方法通用性。
- **消融系统**：对关键设计（算法、任务模仿、风格采样）进行了消融，为实践提供了指导。
- **开源部分代码**：开源了 DMC 人形实验的完整代码，便于复现。

## 八、不足与局限
1. **部分风格在强敌上性能下降**：在 HFW 中，近战、SunScourge 武器、部分元素风格面对高耐久敌人（Apex Thunderjaw、Slaughterspine）胜率低于 50%。
2. **风格奖励设计需要人工精细调参**：权重范围、奖励函数形式依赖专家知识，不具自动适应性。例如零件移除奖励过高导致智能体追逐弱敌直至超时。
3. **风格泛化边界未充分探索**：仅在训练时所见风格参数分布内测试，对于极端超出训练范围的参数（如极大权重），行为可能不可预测。
4. **组合风格证据有限**：虽然展示了组合示例，但仅在少数场景验证，未系统评估所有两两组合的性能。
5. **部分实验统计性不足**：HFW 消融实验仅展示均值区间（5 次运行），没有提供显著性检验。人形域结果主要是示例性展示。
6. **计算资源要求较高**：HFW 单次训练需 3 天单 GPU，难以在资源受限环境中复现。同时未提供总能耗等环境成本数据。
7. **未与最新多目标 RL 或条件策略方法直接对比**：虽然讨论了相关工作的局限，但未在统一基准上比较定量结果（如与 Nauman et al. 2026 对比）。
8. **部分行为符合游戏设计意图但并非最优**：例如零件移除风格中智能体使用爆炸性弹药同时破坏多个部件，属于“粗糙策略”，可能与设计师期望的精确分解不符。
